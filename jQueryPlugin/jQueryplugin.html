နောက် ကျွန်တော်တို့ add လုပ်လိုက်တဲ့ plugin တွေကို responsive ဖြစ်ဖို့ရန်နအတွက် options နှစ်ခုကို plugin ထဲမှာ add လုပ်နိုင်ပါတယ်။ First တစ်ခုကတော့ get width from property ပဲဖြစ်ပါတယ်။ အဲ့ဒါဟာ plugin ကို အဲ့ element ထဲမှာ fixed element ရဲ့ width ကို determines လုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူ့ကို အသုံးပြုပြီးတဲ့နေရာမှာ သူ့ရဲ့ value ဟာ string တစ်ခုဖြစ်ရပါမယ်။ နောက် အဲ့ string ထဲမှာတော့ ကျွန်တော် content တစ်ခုလုံးကိုထည့်ထားတဲ့ CSS selector တစ်ခုဖြစ်သင့်ပါတယ်။ Second ကတော့ responsive width property ကို true ပေးရမှာဖြစ်ပါတယ်။

Code Example

        
            $('.header').sticky({
                getWidthFrom: '.container',
                responsiveWidth: true
            });
            $('.header').on('sticky-start', function(){
                $('.description').html('We build great  apps');
            });
            $('.header').on('sticky-end', function(){
                $('.description').html('We build apps');
            });
            $('.work').sticky({
                topSpacing: 60,
                getWidthFrom: '.container',
                responsiveWidth: true
            });
            $('.work').on('sticky-start', function(){
                $(this).append('<a herf="mailto:email@website.com" class="email-text" ">Email us</a>');
            });
            $('.work').on('sticky-end', function(){
                $('.email-text').remove();
            });

What is Carousel Plugin

jQuery plugins တွေဟာ sophisticated user interfaces တွေကို add လုပ်တဲ့နေရာမှာ quick and simple ဖြစ်ပါတယ်။ အခု section မှာတော့ very popular homepage effect တစ်ခုဖြစ်တဲ့ carousel တစ်ခုကို page မှာ ဘယ်လို add လုပ်ရတယ်ဆိုတာကို လေ့လာသွားမှာဖြစ်ပါတယ်။ Carousels တွေဟာ multiple sides of content တွေကို single slide တစ်ခုထဲမှာ condense လုပ်ပေးတာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ animated slideshow effect ကို create လုပ်ဖို့ရန်အတွက် slides change automatically ပြုလုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့ရဲ့ HTML ထဲမှာဆိုရင် carousel ကို div tag တစ်ခုနဲ့ represent လုပ်တာပဲဖြစ်ပါတယ်။ အဲ့ div တစ်ချင်းစီမှာ any HTML ကို hold လုပ်နိုင်ပါတယ်။ နောက် ကျွန်တော်တို့ဟာ slide တစ်ခုကို navigation system တစ်ခုလို act လုပ်ပေးဖို့ရန်အတွက် links တွေကို add လုပ်နိုင်ပါတယ်။ နောက် section မှာ carousel ကို homepage မှာ add လုပ်သားမှာဖြစ်ပါတယ်။
