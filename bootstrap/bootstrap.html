Introduction to Bootstrap

Bootstrap ဟာဆိုရင် popular ဖြစ်တဲ့ open source front-end frameworks တစ်ခုပဲဖြစ်ပါတယ်။ Bootstrap ဟာဆိုရင် functional design and layout ကို create လုပ်တဲ့နေရာမှာ အကူအညီပေးပါတယ်။ Bootstrap ကိုအသုံးပြုခြင်းရဲ့ biggest advantages တစ်ခုကတော့ development speed ပဲဖြစ်ပါတယ်။ Bootstrap ဟာဆိုရင် web development faster and easier ဖြစ်အောင် လုပ်ပေးပါတယ်။ သူဟာဆိုရင် latest and most advanced front-end tool ကိုအသုံးပြုထားတာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ scratch ကနေပြီးတော့ ကျွန်တော်တို့ရဲ့ site app ကို designing and coding လုပ်မဲ့အစား bootstrap ရဲ့ ready made components ရဲ့ advantage တွေကို အသုံးပြုတာဟာ ပိုပြီးတော့ လွယ်ကူစေပါတယ်။ Bootstrap ဟာဆိုရင် buttons, forms, navigation, grids စတဲ့ common elements တွေအတွက် reusable HTML and CSS ကနေပြီးတော့လာတာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ JavaScript ရဲ့ single line ကို ရေးစရာမလိုတဲ့ dozen custom JavaScrip plugins နဲ့ many of the components တွေကို interactivity ကို add လုပ်တဲ့နေရာမှာ လွယ်လွယ်ကူကူ add လုပ်နိုင်ပါတယ်။ Bootstrap ရဲ့ best part တစ်ခုကတော့ cross-browser compatibility ပဲဖြစ်ပါတယ်။ Bootstrap ကိုအသုံးပြုခြင်းဖြင့် testing လုပ်တဲ့နေရာမှာ အချိန်တွေအများကြီးကုန်နေစရာမလိုတော့ပါဘူး။ သူဟာဆိုရင် testing လုပ်ရမဲ့ အပိုင်းတွေအတွက် already figured out လုပ်ပေးထားတာပဲဖြစ်ပါတယ်။ Bootstrap ကို မလေ့လာခင်မှာ HTML and CSS basic ကို နားလည်ထားဖို့လိုပါတယ်။

Using Bootstrap

Bootstrap ကို 2011 မှာ Twitter မှာရှိတဲ့ developers တွေက create လုပ်ခဲ့တာပဲဖြစ်ပါတယ်။ Bootstrap ဟာ web ပေါ်မှာ building, responsive, mobile first projects တွေအတွက် popular ဖြစ်ပါတယ်။Over 7 million apps and sites တွေကို built လုပ်တဲ့နေရာမှာ bootstrap ကိုအသုံးပြုထားတာပဲဖြစ်ပါတယ်။ Web designers, front end developers and programmers တွေဟာ projects တွေကို build လုပ်တဲ့နေရာမှာ faster and easier ဖြစ်ဖို့ရန်အတွက် Bootstrap ကိုအသုံးပြုကြပါတယ်။ Bootstrap ဟာဆိုရင် front end framework တစ်ခုပဲဖြစ်ပါတယ်။ Framework တစ်ခုဆိုတာ reusable platform တစ်ခုဖြစ်ပြီးတော့ သူဟာ higher level projects ကို build လုပ်တဲ့နေရာမှာ အကူအညီပေးနိုင်ပါတယ်။ Front end framework တစ်ခုဟာ many of the common layout တွေကို solve လုပ်နိုင်ပါတယ်။ Bootstrap ရဲ့ latest documentation ကို getbootstrap.com မှာ လေ့လာနိုင်ပါတယ်။

Setup Starter Template

Bootstrap ကို အသုံးပြုတဲ့နေရာမှာ နည်းလမ်းအနည်းငယ် ရှိပါတယ်။ နည်းလမ်းတစ်ခုချင်းစီဟာ မတူညီတဲ့ skill level or use case တွေကနပြီးတော့ appeal ဖြစ်လာတာပဲဖြစ်ပါတယ်။ Bootstrap ကို getbootstrap.com မှာ download လုပ်နိုင်ပါတယ်။ အဲ့မှာဆိုရင် bootstrap framework ကို ဘယ်လို download လုပ်ရမယ် ဘယ်လို install လုပ်ရမယ်ဆိုတဲ့ instructions တွေကို လုပ်ဆောင်ရမှာဖြစ်ပါတယ်။ Bootstrap ရဲ့ source file တစ်ခုကို new or existing projects ထဲမှာ ထည့်ချင်တဲ့အခါမှာ npm, Bower, even Composer တို့လို package managers တွေကို အသုံးပြုဖို့လိုပါတယ်။ Project တစ်ခုကို bootstrap နဲ့ running လုပ်တဲ့နေရာမှာ fastest way တစ်ခုကတော့ Bootstrap CDN ကိုအသုံးပြုတာပဲဖြစ်ပါတယ်။ CDN ကတော့ cotent delivery network ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ files တွေကို download or install မလုပ်ချင်ဘူးဆိုရင် အဲ့မှာ ရှိတဲ့ CDN links တွေကို အသုံးပြုနိုင်ပါတယ်။ CDN links တွေကို HTML ထဲမှာ add လုပ်တဲ့အခါမှာ CDN servers ကနေ Bootstrap's CSS and JavaScript ကို load လုပ်လိုက်တာဖြစ်ပါတယ်။

Several Componens and Options in Bootstrap

Bootstrap ဟာဆိုရင် project တစ်ခုကို laying out လုပ်ဖို့ရန်အတွက် several components and options တွေရှိပါတယ်။ အဲ့ဒါတွေကတော့ responsive container, grid system, media object and utilities တို့ပဲဖြစ်ပါတယ်။ Containers ဟာ bootstrap ရဲ့ basic layout component တစ်ခုပဲဖြစ်ပါတယ်။ သူဟာ page layout အတွက် foundation တစ်ခုကို provide လုပ်ပေးရုံပဲမဟုတ်ပါဘူး grid system ကို အသုံးပြုတဲ့အချိန်မှာလည်း သူ့ကို လိုအပ်ပါတယ်။ Bootstrap ဟာဆိုရင် mobile အတွက်ပဲ ပထမ ဦးဆုံး develop ဖြစ်လာတာပဲဖြစ်ပါတယ်။ သူ့ရဲ့ media queries ဟာ wider screens and devices တွေအတွက် layout ကို adjust လုပ်ဖို့ရန်အတွက် breakpoints တွေကို create လုပ်ပေးပါတယ်။ သူ့ရဲ့ mobile first approach မှာဆိုရင် bootstrap မှာဆိုရင် five tiers ကို responsive breakpoints အတွက် အသုံးပြုပါတယ်။ Portrait phones or extra small devices မှာဆိုရင် default range ဟာဆိုရင် layout အတွက်ပဲဖြစ်ပါတယ်။ အဲ့ media query range ဟာဆိုရင် landscape phones, tablets, desktops and large desktops တွေအတွက်ပဲဖြစ်ပါတယ်။ Bootstrap ဟာဆိုရင် သူ့ရဲ့ media queries အတွက်ဆိုရင် pixel units ကိုအသုံးပြုပါတယ်။ Extra small range အတွက်ဆိုရင် less than 576 pixels ကိုအသုံးပြုရပါတယ်။ Small range ဆိုရင် 576 pixels and up ကိုအသုံးပြုပါတယ်။ Medium range ဆိုရင် 768 pixels and up ကိုအသုံးပြုပါတယ်။ Large range ဆိုရင် 992 pixels and up ကိုအသုံးပြုပါတယ်။ Extra large range အတွက်ဆိုရင် 1200 pixels ကိုအသုံးပြုပါတယ်။

Bootstrap Grid System

အခု bootstrap ရဲ့ grid system ကိုအသုံးပြုပြီးတော့ page ကို laying out လုပ်မှာဖြစ်ပါတယ်။ Example မှာဆိုရင် bootstrap's CDN links ကို အသုံးပြုထားပါတယ်။ ပြီးတော့ bootstrap plugins အားလုံးဟာ proper.js ဆိုတဲ့ jQuery နဲ့ JavaScript library ကိုအသုံးပြုထားပါတယ်။ Bootstrap ရဲ့ mobile-first flexbox grid ဟာဆိုရင် shapes and sizes အားလုံးရဲ့ layouts ကို build လုပ်ဖို့ရန်အတွက် 12 columns system ကိုအသုံးပြုပါတယ်။ Bootstrap's grid system မှာဆိုရင် major components ၃ ခုရှိပါတယ်။ အဲ့ဒါတွေကတော့ containers, rows and columns တို့ပဲဖြစ်ပါတယ်။ Example မှာဆိုရင် အဲ့ components ၃ ခုလုံးဟာ ဘယ်လို အတူတူအလုပ်လုပ်တယ်ဆိုတာကို လေ့လာနိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ container ထဲမှာ rows တွေကို add လုပ်လို့ရပါတယ်။ နောက် row ထဲမှာ columns တွေကို add လုပ်နိုင်ပါတယ်။ Columns တွေဟာဆိုရင် rows တွေအတွက် immediate children တွေဖြစ်တဲ့အတွက်ကြောင့် သူတို့ဟာ content ထဲမှာ contain ဖြစ်နေတာပဲဖြစ်ပါတယ်။ Rows တစ်ခုတိုင်းမှာ 1 to 12 columns တွေပါရှိနိုင်ပါတယ်။ Column တစ်ခုချင်းစီကို classes တွေနဲ့ assign လုပ်နိုင်ပါတယ်။ ပြီးတော့ classes name နဲ့ width ကို specifies လုပ်နိုင်ပါတယ်။ HTML comment တစ်ခုကို create လုပ်ဖို့ဆိုရင် cmd or ctrl နဲ့ forward slash ကို နှိပ်ပေးရမှာဖြစ်ပါတယ်။

Scrollspy in Bootstrap

Bootstrap မှာဆိုရင် Scrollspy လုပ်ဖို့ရန်အတွက် simple solution တစ်ခုရှိပါတယ်။ Scrollspy ဟာ interactive JavaScript plugin တစ်ခုပဲဖြစ်ပါတယ်။ သူဟာ active navigation links ကို highlights လုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူ့ကို user ရဲ့ scroll position ပေါ်မှာမူတည်ပြီးတော့ nav links တွေကို active လုပ်သွားမှာဖြစ်ပါတယ်။ အဲ့ဒါကို create လုပ်ဖို့ဆိုရင် data-spy attribute တစ်ခုကို page ရဲ့ body ထဲမှာ add လုပ်ပေးရမှာဖြစ်ပါတယ်။ ပြီးရင် data-target attribute ကို id or ကျွန်တော်တို့ navigation component ရဲ့ class name နဲ့ add လုပ်ပေးရမှာဖြစ်ပါတယ်။
