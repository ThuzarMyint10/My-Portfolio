About HTML and CSS

HTML and CSS တို့ဟာ web page တစ်ခုကို သူတို့ရဲ့ code တွေနဲ့ apply ဖြစ်စေတာပဲဖြစ်ပါတယ်။ Coding တွေရဲ့ vocabulary ဟာ learn လုပ်တဲ့အချိန်မှာ အရေးကြီးပါတယ်။ Code တွေကို မပြောခင်မှာ HTML နဲ့ CSS တို့အကြောင်းကို အရင်ဦးဆုံးပြောပြချင်ပါတယ်။ HTML and CSS တို့ဟာ website တိုင်းရဲ့ အခြေခံကျတဲ့ building block တွေပဲဖြစ်ပါတယ်။ HTML ဟာ home တစ်ခုရဲ့ carpenter တွေဖြစ်ပြီးတော့ CSS ကတော့ interior designer ပဲဖြစ်ပါတယ်။ သူကတော့ home ရဲ့ selecting color schemes, window treatment, carpet, and lighting တွေကိုလုပ်ဆောင်ပေးတာပဲဖြစ်ပါတယ်။ HTML ဟာ webpage တိုင်းရဲ့ structure foundation ပဲဖြစ်ပါတယ်။ အဲ့နေရာမှာ languages တွေအများကြီးရှိပါတယ်။ CSS ကတော့ webpage တွေရဲ့ fonts, color, and page layout တွေကို modify လုပ်ပေးတာပဲဖြစ်ပါတယ်။

About Comments and Doctype

အခုဆိုရင် ကျွန်တော်တို့ဟာ HTML နဲ့ CSS တို့နဲ့ familiar ဖြစ်နေပါပြီ။ ကျွန်တော်အရင်ဆုံးပြောပြချင်တာကတော့ comment အကြောင်းပဲဖြစ်ပါတယ်။ အဲ့ comment line ကို တော့ code တွေကို run တဲ့အခါမှာ ပြမှာမဟုတ်ပါဘူး။ HTML code မှာ comments တွေကို create လုပ်တဲ့အခါမှာ less than sign ရယ် exclamation point and two dashes ရယ်ပါရပါမယ်။ Close comment ကိုတော့ ရှေ့က two dashes နှစ်ခုနဲ့ နောက်ကgreater than sign ကိုရေးပေးရပါမယ်။ Comment ကို ကျွန်တော်တို့်ရဲ့ code တွေရဲ့ အကြောင်းအရာကိုဖော်ပြချင်တဲ့အခါအသုံးပြုနိုင်ပါတယ်။ နောက်ထပ်ကျွန်တော်ပြောချင်တာကတော့ document type လို့ခေါ်တဲ့ doc type အကြောင်းပဲဖြစ်ပါတယ်။ Doctype ဟာဆိုရင် document ဟာ ဘာအမျိုးအစားဖြစ်တယ်ဆိုတာကို browser ကိုပြောပြပေးတာပဲဖြစ်ပါတယ်။ ဒါဟာ HTML document မှာ အရေးကြီးတဲ့ thing တစ်ခုပဲဖြစ်ပါတယ်။ နောက်တစ်ခုကတော့ HTML tag ပဲဖြစ်ပါတယ်။ အဲ့ဒါကို instructions အနေနဲ့ အသုံးပြုနိုင်ပါတယ်။ Instructions ကဘာလို့ဆိုရင် သူလိုပဲ ကျန်တဲ့ tags တွေမှာလည်း angle brackets တွေ ရှိရယ်ဆိုတာပဲဖြစ်ပါတယ်။ Tags တွေကို close လုပ်တဲ့အခါမှာ "/" လေးကို less than sign ရဲ့နောက်ကထည့်ပေးရမှာဖြစ်ပါတယ်။

About Head

Doc type ရဲ့အောက်မှာ ကျွန်တော်တို့ဟာ head tag ကိုရေးပေးရပါမယ်။ သူထဲမှာဆိုရင် document ရဲ့ meta information တွေပါဝင်ပါတယ်။ HTML page တစ်ခုဟာ humans နဲ့ similar ဖြစ်ပါတယ်။ Meta information ဟာ meta cognition နဲ့အတူတူပဲဖြစ်ပါတယ်။ Meta tag ထဲမှာဆိုရင် meta charset UTF-8 ဆိုတဲ့ line ကိုတွေ့ရပါလိမ့်မယ်။ Charset ဆိုတာကတော့ character set ကိုဆိုလိုပါတယ်။ Websites တွေ create လုပ်တဲ့အခါမှာ UTF-8 ကို 99% သုံးကြပါတယ်။ နောက်တစ်ခုကတော့ title tag ပဲဖြစ်ပါတယ်။ Title tag ဟာ browser bar မှာ ပေါ်လာတဲ့ title ကိုရေးတဲ့အခါမှာအသုံးပြုပါတယ်။ နောက်တစ်ခုကတော့ style tag ပဲဖြစ်ပါတယ်။ သူကိုတော့ CSS file ကိုယ်စား အသုံးပြုနိုင်ပါတယ်။ သူထဲမှာ CSS file လိုပဲ ကျွန်တော် apply ချင်တဲ့ကောင်တွေကို properties တွေကိုသုံးပေးရမှာဖြစ်ပါတယ်။

About Body

Head tag ရဲ့အောက်မှာတော့ document ရဲ့ main content တွေကို ရေးရတဲ့ body element ပဲဖြစ်ပါတယ်။ အဲ့ element မှာဆိုရင် browser မှာမြင်နိုင်ရတဲ့ အရာတွေအားလုံးကိုရေးရတဲ့ အပိုင်းပဲဖြစ်ပါတယ်။ ဥပမာ text, links to other Web pages, images, tables and lists တို့ပဲဖြစ်ပါတယ်။
