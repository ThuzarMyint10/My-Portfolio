Overview of Tables

ကျွန်တော်တို့အခု HTML tables ကိုအသေးစိတ်လေ့လာကြည့်ကြစို့။ HTML table ဆိုတာ tabular data လို့ခေါ်တဲ့ information တွေကို display လုပ်ပေးတာပဲဖြစ်ပါတယ်။ အဲ့ information တွေကို table တွေကို columns တွေ rows တွေနဲ့ stored လုပ်ထားတာဘဲဖြစ်ပါတယ်။ Table ဟာ paragraphs and images တို့ထက်အရင်ပေါ်လာတာဖြစ်ပါတယ်။ ဘာကြောင့်လဲဆိုရင် table တစ်ခုဆောက်တဲ့နေရာမှာ ကျွန်တော်တို့ဟာ table cells တစ်ခုချင်းစီနဲ့ အဲ့ cells တွေပါဝင်တဲ့ rows တွေကို represent လုပ်ဖို့ရန်အတွက် elements တွေကိုအသုံးပြုဖို့လိုပါတယ်။ CSS မှာဆိုရင် table ကို create လုပ်တာဟာ browser defaults ထက်ပိုပြီးကောင်းပါတယ်။

How to create a Basic Table

ကျွန်တော်အခု Table တစ်ခုကို ဘယ်လို create လုပ်တယ်ဆိုတာနဲ့ သူ့ထဲကို rows တွေ cells တွေကို ဘယ်လို put လုပ်တယ်ဆိုတာကို ပြောပြပေးသွားမှာ ဖြစ်ပါတယ်။ Table တစ်ခုကို create လုပ်တဲ့ အခါ <table> tag ကို အသုံးပြုရမှာ ဖြစ်ပါတယ်။ <table> tag ထဲမှာ attribute တွေထည့်သုံးနိုင်ပါတယ်။ အဲ့ဒါတွေကတော့ border, rules, bgcolor, cellspacing, etc. အသုံးပြုနိုင်ပါတယ်။ Table ထဲကို rows တွေထည့်ချင်ရင် <tr> tag ကို အသုံးပြုရမှာဖြစ်ပါတယ်။ အဲ့ <tr> tag ထဲကို data or cells တွေကို <td> tag ကို သုံးပေးရပါမယ်။ အဲ့ tag တွေဘယ်လိုအလုပ်လုပ်တယ်ဆိုတာကို အောက်က example မှာလေ့လာနိုင်ပါတယ်။

How to create Table Header and Cell elements

အခုကျွန်တော်ပြောချင်တဲ့အကြောင်းလေးကတော့ Tables တွေအကြောင်းပဲဖြစ်ပါတယ်။ Tables တွေအကြောင်းကို သိသင့်သလောက်သိထားပြီဖြစ်ပါတယ်။ ဒီတစ်ခါမှာတော့ ကျွန်တော်တို့ဟာ labeling name ကို table header cells တွေလို့ထင်နိုင်ပါတယ်။ ဘာကြောင့်လဲဆိုရင် သူတို့ဟာ labeling columns လည်းမဟတ်သလို labeling rows တွေလည်းမဟုတ်တဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ Table header ဟာဆိုရင်အမှန်တစ်ကယ် labeling ဖြစ်နေတဲ့ row တစ်ခု(သို့) column တစ်ခုပဲဖြစ်ပါတယ်။ Columns တွေဟာ rows တစ်ခုချင်းစီးရဲ့ header ဖြစ်နိုင်ပါတယ်။ ဒီမှာဆိုရင် scope attribute ကိုအသုံးပြုပါမယ်။ အဲ့ attribute ဟာ header ဟာ row နဲ့ လည်းတူသလို column နဲ့လည်းအတူတူဖြစ်တယ်ဆိုတာကို ဖော်ပြတာပဲဖြစ်ပါတယ်။ကျွန်တော်ပြောပြခဲ့တာတွေကို အောက်က example မှာလေ့လာနိုင်ပါတယ်။

About Table Head and Body

အခုတစ်ခါ Table နဲ့ ပါတ်သတ်ပြီး ရှင်းပြပေးသွားမှာကတော့ Table ရဲ့ header အကြောင်းနဲ့ Table ရဲ့ body အကြောင်းပဲဖြစ်ပါတယ်။ <thead> tag ကို table ရဲ့ head အတွက်အသုံးပြုပေးရမှာဖြစ်ပါတယ်။ Table ရဲ့ body အတွက်ကတော့ <tbody> tag ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ <thead> tag ကိုသုံးတဲ့ အခါမှာ အဲ့ tag သုံးထားတဲ့ rows ဟာ တစ်ခြား rows တွေနဲ့ မတူဘဲ highlight ပြပေးမှာဖြစ်ပါတယ်။ &tl;tbody> tag ကိုတော့ table ရဲ့ data တွေအတွက်အသုံးနိုင်ပါတယ်။ အဲ့လိုအသုံးပြုတာဟာ table ထဲမှာရှိတဲ့ data တွေကို ဘယ် data တွေက head ဖြစ်ပြီးတော့ ဘယ် data တွေက table ရဲ့ body ဖြစ်တယ်ဆိုတာကို သိရှိစေမှာဖြစ်ပါတယ်။ အောက်က example မှာ သူတို့ရဲ့ အသုံးပြုပုံကို လေ့လာကြည့်နိုင်ပါတယ်။

About Table Footer Element

Table ထဲ foot element ဟာဘယ်လို effect ဖြစ်တယ်ဆိုတာကို ပြောပြပေးမှာဖြစ်ပါတယ်။ Footer element ဆိုတာ table ရဲ့ summary ပဲဖြစ်ပါတယ်။ ပြောချင်တာကတော့ သူဟာ column တစ်ခုချင်စီရဲ့ sums, totals, and averages တွေကို ဖော်ပြပေးတာပဲဖြစ်ပါတယ်။ A table footer မှာဆိုရင် table နဲ့ ပါတ်သတ်တဲ့ meta-information တွေပါဝင်ပါတယ်။ သူဟာ colspan attribute မပါဘဲအသုံးပြုရင် သူဟာ column တစ်ခုထဲကိုပဲ apply လုပ်မှာဖြစ်ပါတယ်။ span attribute ကိုသုံးမှသာ columns အားလုံးကို apply လုပ်မှာဖြစ်ပါတယ်။ အဲ့ colspan attribute ထဲမှာ ကိုယ် apply လုပ်စေချင်တဲ့ column အရေးအတွက်ကိုထည့်ပေးရမှာဖြစ်ပါတယ်။

How to use Table

အခုတစ်ခါကျွန်တော်ပြောပြပေးမှာကတော့ table တွေကို ဘယ်လိုနေရာတွေမှာအသုံးပြုတယ် ဆိုတာဘဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ table တွေကို web browser တစ်ခုမှာ layout အတွက် ဘယ်တော့မှာအသုံးပြုလို့မရပါဘူး။ Table တွေကို tabular data တွေကို display လုပ်ဖို့အတွက်သာအသုံးပြုနိုင်ပါတယ်။ သူ့ဟာ CSS နဲ့ တွဲပြီး အလုပ်လုပ်နိုင်သောကြောင့် သူ့ကို images and text တွေကို positioning လုပ်ဖို့အတွက် အသုံးမပြုနိုင်ပါတယ်။ HTML ရဲ့ emails တွေ tables တွေမှာ layout အတွက် အသုံးပြုပါတယ်။ တစ်ကယ်လို့ front end development ကို လေ့လာတဲ့အခါမှာ table ထဲကို jQuery အချို့ကို add လုပ်ပေးဖို့လိုပါတယ်။ အဲ့လိုလုပ်တဲ့နေရာမှာ right plug-in နဲ့ အခြား static table ထဲကို sorting and filtering လို features တွေကို add လုပ်နိုင်ပါတယ်။ တစ်ကယ်လို့ Ruby, PHP and Python တု့ိလို back end languages တွေကို လေ့လာတဲ့ အခါမှာ database တစ်ခုကနေ data တွေကို အသုံးပြုပြီး table တစ်ခုကို populate လုပ်နိုင်ပါတယ်။ Database ဆိုတာ table structure တစ်ခုထဲမှာ information တွေကို store လုပ်ထားတာဘဲဖြစ်ပါတယ်။ အဲ့ဒါကြောင့် database တစ်ခုက data တွေကို အလွယ်တစ်ကူမြင်နိုင်ပါတယ်။
