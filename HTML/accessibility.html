ကျွန်တော်တို့မှာ visual and hearing impairments အကြောင်းကို ပြောပြခဲ့ပြီးဖြစ်ပါတယ်။ အခုနောက်ထပ် other impairments တွေနဲ့ ကျွန်တော်တို့ဟာ accessibility ကို ဘယ်လို improve ဖြစ်အောင်လုပ်နိုင်တယ်ဆိုတာကို လေ့လာကြမှာဖြစ်ပါတယ်။ လူတွေဟာ web accessibility အကြောင်းကို ပြောကြတဲ့အခါမှာ visual and hearing impairments ကို အရင်းဆုံးစဉ်းစားကြတာပဲဖြစ်ပါတယ်။ သို့သော်လည်း impairments တွေအများကြီးရှိနိုင်ပါသေးတယ်။ အဲ့ဒါတွေကတော့ fine motor skills and muscle control တွေ ပြီးတော့ muscle movement ကို slow ဖြစ်စေနိုင်ပါတယ်။ ဒါဟာ challenging တစ်ခုပဲဖြစ်ပါတယ်။ အဲ့ဒါဟာ impaired mobility ရှိတဲ့ users တွေအတွက် ကျွန်တော်တို့ရဲ့ sites ကို ပိုပြီးတော့ accessible ဖြစ်အောင်ပြုလုပ်နိုင်ပါတယ်။ First အနေနဲ့ကတော့ ကျွန်တော်တို့ရဲ့ site keyboard ပေါ်မှာ functionality တွေကို accessible ဖြစ်အောင်ပြုလုပ်တာပဲဖြစ်ပါတယ်။ ဒါဟာ အလွန်လွယ်ကူပါတယ် ဘာလို့လဲဆိုရင် browsers and operating systems တွေိမှာဆိုရင် web pages တွေအတွက် robust keyboard navigation ပါတဲ့အတွက်ြကောင့်ပဲဖြစ်ပါတယ်။ နောက်တစ်ခုကတော့ ကျွန်တော်တို့ဟာ users တွေရဲ့ task ကို accomplish ဖြစ်ဖို့ရန်အတွက် လုံလောက်တဲ့ time ကို provide လုပ်ပေးရပါမယ်။ Static web page တစ်ခုမှာဆိုရင် အဲ့ဒါကို လုပ်ဆောင်ပြီးသားဖြစ်ပါတယ်။ နောက်ဆုံးတစ်ခုကတော့ ကျွန်တော်တို့ဟာ number of steps တွေကို minimize ဖြစ်အောင် လုပ်သင့်ပါတယ်။ ဒါဟာဘာကို ဆိုလိုတာလဲဆိုရင် special keyboard shortcuts ကို creating လုပ်တယ်၊ ကျွန်တော်တို့ရဲ့ user interface ကို simplifying လုပ်ပေးတယ်၊ amount of clicking ကို cutting down လုပ်တာတွေပဲဖြစ်ပါတယ်။

Screen Reader

ကျွန်တော်တို့ဟာ improving accessibility အတွက် some techniques အကြောင်းတွေကို လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခုကျွန်တော်တို့ဟာ real project ကို apply လုပ်မှာဖြစ်ပါတယ်။ ကျွန်တော်တို့်ဟာ screen readers နဲ့ အနည်းငယ် familiar ဖြစ်ပြီးသားဖြစ်ပါတယ်။ Screen reading software ရဲ့ pieces တွေအများကြီးရှိပါတယ်။ တစ်ခုကတော့ NVDA(NonVisual Desktop Access) ပဲဖြစ်ပါတယ်။ အဲ့ဒါဟာ web site တစ်ခုပဲဖြစ်ပါတယ်။ ဒီ software ဟာဆိုရင် window မှာပဲ အလုပ်လုပ်တာပဲဖြစ်ပါတယ်။ နောက်တစ်ခုကတော့ JAWS ဟာဆိုရင် popular choice တစ်ခုပဲဖြစ်ပါတယ်။ အဲ့ဒါဟာ free မဟုတ်ပါဘူး။ သူ့ကို $1000 ပေးရပါတယ်။ သူဟာလည်း window မှာပဲ အလုပ်လုပ်တာပဲဖြစ်ပါတယ်။ နောက်ဆုံးတစ်ခုကတော့ VoiceOver ပဲဖြစ်ပါတယ်။ သူဟာဆိုရင်တော့ Mac OS X မှာပဲ အလုပ်လုပ်တာပဲဖြစ်ပါတယ်။ Screen reader ဟာ "link" လို့လဲပြောနိုင်ပါတယ်။ သူဟာ page ရဲ့ single link တစ်ခုပဲဖြစ်ပါတယ်။
