Sorting Multidimensional Arrays

အခု ကျွန်တော်တို့ next feature တစ်ခုဖြစ်တဲ့ ability တစ်ခုကတော့ sort လုပ်ဖို့ရန် filed ကိုအသုံးပြုတာပဲဖြစ်ပါတယ်။ အဲ့အတွက် ကျွန်တော်တို့ရဲ့ filter array မှာ values နှစ်ခု ပါဝင်ဖို့လိုပါတယ်။ နောက် ကျွန်တော်တို့ရဲ့ list array ထဲမှာအသုံးပြုထားတဲ့ ID ကိုလည်း store လုပ်ဖို့ရန်လိုပါသေးတယ်။ ဒါပေမဲ့ ကျွန်တော်တို့ sorting အတွက် အသုံးပြုဖို့ရန်အတွက် information တွေကိုလည်း လိုအပ်ပါတယ်။ ကျွန်တော်တို့ PHP ကနေ ကျွန်တော်တို့ရဲ့ filter array အတွက် key တစ်ခုကို automatically to produce လုပ်တာအစား ကျွန်တော်တို့ဟာ own key value pair ကို set လုပ်နိုင်ပါတယ်။ အဲ့လို set လုပ်တဲ့နေရာမှာ key ဟာ unique ဖြစ်ရပါမယ်။ ကျွန်တော်တို့ဟာ same key ကို အသုံးပြုမယ်ဆိုရင် သူဟာ existing value ကို override လုပ်သွားမှာဖြစ်ပါတယ်။
