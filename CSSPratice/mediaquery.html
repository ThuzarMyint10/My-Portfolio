Practice Media Query

ဒီတစ်ခါမှာတော့ CSS practice ထဲက media query အကြောင်းကို လေ့လာသွားမှာဖြစ်ပါတယ်။ Media query ဟာဆိုရင် web design and front end web development ရဲ့ important part တစ်ခုပဲဖြစ်ပါတယ်။Media query ဟာဆိုရင် content ကို specific range of devices နဲ့ screen sizes တွေမှာ content တွေဟာ သူတို့ကိုယ်တိုင် change မလုပ်ဘနဲ့ presentation လုပ်ပေးတာပဲဖြစ်ပါတယ်။ ဒီ practice session မှာဆိုရင် simple web page တစ်ခုရဲ့ layout ကို media queries တွေရေးပြီးတော့ adjust လုပ်သွားမှာဖြစ်ပါတယ်။ Example မှာ လေ့လာနိုင်ပါတယ်။

Code in CSS

        
                /* ================================================= 
                Practice Media Queries
             ==================================================== */
             
             /* Target viewport sizes less than 576px */
             
             @media (max-width: 575px) {
               header {
                 text-align: center;
                 margin-bottom: 2.5rem;
               }
               .logo {
                 margin-bottom: 0;
               }
               nav a {
                 padding: 6px 0;
                 background: #def1f9;
                 margin: .2em 0;
               }
               img {
                 display: none;
               }
             }
             
             /* Target viewport sizes 576px and wider */
             
             @media (min-width: 576px) {
               .logo {
                 float: left;
               }
               nav {
                 margin-top: 24px;
               }
               nav li {
                 display: inline-block;
                 margin: 0 0.45em;
               }
               h2 {
                 margin-bottom: 1.5em;
               }
             }
             
             /* Target a viewport range wider than 575px and narrower than 992px */
             
             @media (min-width: 576px) and (max-width: 991px) {
               nav {
                 float: left;
                 margin-left: 20px;
               }
             }
             
             /* Target viewport sizes 768px and wider */
             
             @media (min-width: 768px) {
               .articles {
                 float: left;
                 width: 65%;
               }
               aside {
                 float: right;
                 width: 30%;
               }
             
               img {
                 width: 100%;
               }
             }
             
             /* Target viewport sizes 992px and wider */
             
             @media (min-width: 992px) {
               nav {

               }

Blend Modes

ကျွန်တော်တို့အခု လေ့လာမှာကတော့ favorite graphics editor တွေမှာ မရှိတဲ့ resembles tool ဖြစ်တဲ့ blend modes လို့ခေါ်တဲ့ exciting CSS feature တွေကို ဘယ်လိုအသုံးပြုတယ်ဆိုတာကို လေ့လာသွားမှာဖြစ်ပါတယ်။ CSS blend modes ဟာဆိုရင် two or more layers ကို ဘယ်လို blend လုပ်တယ်ဆိုတာကို define လုပ်ပေးတာဖြစ်ပါတယ်။ သူဟာဆိုရင် really exciting visual effects တွေကို create လုပ်ပေးပါတယ်။ Blending effects တွေကို graphic design or digital image editing လုပ်တဲ့နေရာမှာ အသုံးပြုကြပါတယ်။ CSS blend modes ကိုအသုံးပြုပြီးတော့ browser မှာ same blending effects တွေကို recreate လုပ်နိုင်ပါတယ်။ Blend modes ဟာ compositing and blending CSS specification ရဲ့ part ပဲဖြစ်ပါတယ်။ Compositing ဆိုတာ element တစ်ခုနဲ့ သူ့ရဲ့ backdrop ကို combine လုပ်တာပဲဖြစ်ပါတယ်။ Blending ဆိုတာ colors တွေရဲ့ mixing ကို calculates လုပ်တဲ့ compositing ရဲ့ form တစ်ခုပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ အဲ့ဒါတွေကို အသုံးပြုပြီးတော့ HTML element တစ်ခုရဲ့ background layer or another layer ကို blend လုပ်နိုင်ပါတယ်။

CSS မှာ ဆိုရင် elements တွေကို blend လုပ်ဖို့ရန်အတွက် properties နှစ်ခုရှိပါတယ်။ Property တစ်ခုချင်းစီမှာ blend modes 16 ရှိပါတယ်။ Properties တွေကတော့ background-blend-mode နဲ့ mix-blend-mode တို့ပဲဖြစ်ပါတယ်။ Background-blend-mode property ကို နှစ်ခု (သို့) နှစ်ခုထက်ပိုတဲ့ background layers တွေကို blend လုပ်ချင်တဲ့အခါမှာအသုံးပြုနိုင်ပါတယ်။ Mixed-blend-mode property ကို any HTML or SVG တို့ရဲ့ content တွေနဲ့ blend လုပ်နိုင်ပါတယ်။ Background blend mode property ဟာ element တစ်ခုရဲ့ background image နဲ့ သူ့ရဲ့နောက်ထပ် image or background color တို့ကို blend လုပ်တဲ့နေရာမှာ အသုံးများပါတယ်။ Background-blend-mode property နဲ့blend mode value ကို specify လုပ်တဲ့အခါမှာ သူဟာ background image နဲ့ color ကို ကျွန်တော်တို့ set လုပ်လိုက်တဲ့ blend mode ပေါ်မှာ အခြေခံပြီးတော့ mix လုပ်ပေးတာပဲဖြစ်ပါတယ်။ Background-blend-mode property ရဲ့ default value ကတော့ normal ပဲဖြစ်ပါတယ်။ နောက်ထပ် သုံးနိုင်တဲ့ values တွေရှိပါသေးတယ်။ ကျန်တဲ့ values တွေကို web platform docs မှာ သွားရောက်လေ့လာနိုင်ပါတယ်။ Mix-blend-mode property မှာဆိုရင်လည်း values 16 ခုရှိပါတယ်။ အဲ့ဒါတွေကို web platform docs မှာလေ့လာနိုင်ပါတယ်။

          html, 
          body {
              height: 100%;
          }
          
          div {

          }

Masking

အခု exciting CSS design effect တစ်ခုဖြစ်တဲ့ CSS mask အကြောင်းကို လေ့လာသွားမှာဖြစ်ပါတယ်။ Masking and clipping တို့ဟာ photoshop and sketch တို့လို graphic editing tools တွေမှာ အသုံးများတဲ့ features တစ်ခုပဲဖြစ်ပါတယ်။ သူတို့ကို element တစ်ခုရဲ့ parts တွေကို hide လုပ်ဖို့ရန်အတွက်အသုံးပြုနိုင်ပါတယ်။ CSS ကို အသုံးပြုပြီးတော့ masks ကို browser မှာ direct apply လုပ်နိင်ပါတယ်။ Masking ဟာ CSS masking specification ရဲ့ part ပဲဖြစ်ပါတယ်။ သူဟာ HTML elements တွေရဲ့ hide ထားတဲ့ portions တွေကို partially or fully အတွက် methods နှစ်ခုကို provide လုပ်ပေးပါတယ်။ Element တစ်ခုကို masking လုပ်တဲ့နေရာမှာ element ရဲ့ visual portions ကို filter လုပ်ဖို့ရန်အတွက် mask image တစ်ခုကို အသုံးရန်လိုအပ်ပါတယ်။

Code in CSS

      
          * {
            box-sizing: border-box;
        }
        html,
        body,
        .wrap {
            height: 100%;
            margin: 0;
        }
        .wrap {
            background: url('mask.jpg') no-repeat;
        }
        
        h1 {
            font-family: Arial, Helvetica, sans-serif;
            color: #fff;
            margin: 0;
            font-size: 10em;
            text-shadow: 2px 2px 5px rgba(0, 0, 0, .4);
            -webkit-mask-image: url('texture1.jpg');
            mask-image: url('texture1.jpg');
            padding: 60px 0 0 200px;
        }
