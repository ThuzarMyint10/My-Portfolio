လုပ်ထားတာပဲဖြစ်ပါတယ်။ နောက် Enzyme ဟာ React အတွက် JavaScript testing utility တစ်ခုဖြစ်ပါတယ်။ သူ့ကို Airbnb ကနေ created and used လုပ်တာဖြစ်ပါတယ်။ သူကို other testing libraries တွေနဲ့ compatible ဖြစ်အောင်လည်း built လုပ်ထားတာဖြစ်ပါတယ်။ Unit test ကို write လုပ်ခြင်းဖြင့် code တွေမှာ improve ဖြစ်အောင် ကူညီပေးနိုင်ပါတယ်။

Developers Roles

Typical software development team တစ်ခုမှာဆိုရင် several different roles တွေနဲ့ ဖွဲ့စည်းထားတာပဲဖြစ်ပါတယ်။ Deverlopers တွေဟာ code တွေကို ရေးရတာဖြစ်ပြီးတော့ design the user interface, test and report bugs, prototype, manage the team and more တို့ပါဝင်ပါတယ်။ အခုပြောပြပေးမှာကတော့ development team တစ်ခုမှာရှိတဲ့ some of the roles developers တွေနဲ့ နေ့တိုင်းလုပ်ဆောင်ရတဲ့အလုပ်အကြောင်းတွေပဲဖြစ်ပါတယ်။ JavaScript developers or most developers မှာ junior developer and senior developer ဆိုပြီးတော့ roles နှစ်ခုရှိပါတယ်။ Junior and senior developers နှစ်ခုကို လုံးဟာ developer community ကို ထောက်ပံ့ပေးကြပါတယ်။ Development teams အားလုံးကတော့ roles တွေကို set up လုပ်တဲ့ နေရာမှာ မတူကြပါဘူး။ Teams တွေကို company ရဲ့ လိုအပ်ချက်နဲ့ fit ဖြစ်အောင် create လုပ်ထားကြပါတယ်။ Company or project ပေါ်မှာမူတည်ပြီးတော့ architects, designers, product managers, QA testers, and more စတဲ့ other roles တွေမှာအလုပ်လုပ်ကြရပါတယ်။
