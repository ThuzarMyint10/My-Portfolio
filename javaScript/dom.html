DOM or Document Object Model

ပြီးခဲ့တဲ့ section မှာ document object အကြောင်းကို ပြောပြခဲ့ပြီးဖြစ်ပါတယ်။ Document ဟာဆိုရင် HTML and content of a webpage တို့ကို representing လုပ်ပေးတဲ့ global object တစ်ခုပဲဖြစ်ပါတယ်။ Document ကိုအသုံးပြုပြီးတော့ webpage ရဲ့ different parts တွေကို select and change လုပ်နိုင်ပါတယ်။ အဲ့ဒါတွေဘယ်လို အလုပ်လုပ်တယ်ဆိုတာကို နားလည်ထားတာဟာ DOM ရဲ့ understanding တစ်ခုကို နားလည်တာပဲဖြစ်ပါတယ်။ DOM ကတော့ document object model ပဲဖြစ်ပါတယ်။ DOM ဆိုတာ JavaScript ကနေအသုံးပြုနိုင်တဲ့ webpage တစ်ခုရဲ့ representation တစ်ခုပဲဖြစ်ပါတယ်။ DOM ဟာ web page တစ်ခုကို tree-like structure တစ်ခုအဖြစ်နဲ့ represent လုပ်ပေးပါတယ်။ HTML web page တစ်ခုမှာဆိုရင် head တစ်ခု body တစ်ခုရှိပါတယ်။ နောက် သူတို့ထဲမှာ nested ဖြစ်နေတဲ့ အခြား elements တွေလည်းရှိတယ်ဆိုပါတော့ အဲ့မှာဆိုရင် document element or node ဟာ tree ရဲ့ root node တစ်ခုဖြစ်ပါတယ်။ Head နဲ့ body ကတော့ အခြား branches တွေကို leading လုပ်တဲ့ branches တွေပဲဖြစ်ပါတယ်။ Tree ရဲ့ top မှာရှိတဲ့ leaves တွေဟာ heading element or list item nodes တွေပဲဖြစ်ပါတယ်။

Another Action

ဒီ section မှာဆိုရင် browser environment မှာရှိတဲ့ global scope အကြောင်းကို လေ့လာသွားမှာဖြစ်ပါတယ်။ JavaScript တွေ load မလုပ်ခင်မှာကတည်းက a lot of global variables တွေရှိပါတယ်။ Main global variable ဟာတော့ document object ပဲဖြစ်ပါတယ်။ သူဟာဆိုရင် other global variables တွေနဲ့အတူတူပဲဖြစ်ပါတယ်။ နောက်သူဟာ window object ရဲ့ property တစ်ခုလည်းဖြစ်ပါတယ်။ အစောပိုင်းမှာ element တစ်ခုကို JavaScript နဲ့ controlling လုပ်တဲ့ two step အကြောင်းကိုပြောပြခဲ့ပြီးဖြစ်ပါတယ်။ First, element ကို select လုပ်ပြီးတော့ read or manipulate လုပ်နိုင်ပါတယ်။ အခု another action ကိုအပြောပြပေးမှာဖြစ်ပါတယ်။ အဲ့ဒါကတော့ element တစ်ခုကို some event တစ်အတွက် listen လုပ်ဖို့ရန် ask လုပ်နိုင်ပါတယ်။ Event ဆိုတာကတော့ click or key press တို့လို user action ပဲဖြစ်ပါတယ်။

Code Example

        
                const myHeading = document.getElementById('myHeading');

                myHeading.addEventListener('click', () => {
                    myHeading.style.color = 'red';
                
                });

Select an Element By TagName

ပြီးခဲ့တဲ့ section မှာ element တစ်ခုကို ID နဲ့ select လုပ်ခဲ့တာဖြစ်ပါတယ်။ သူဟာ single element တစ်ခုကိုပဲ select လုပ်ပေးတာဖြစ်ပါတယ်။ အခု section မှာတော့ ကျွန်တော်တို့ဟာ multiple elements တွေကို select လုပ်ချင်တဲ့အခါမှာ document.getElementsByTagName ဆိုတဲ့ method ကိုအသုံးပြုပေးရမှာဖြစ်ပါတယ်။ သူဟာ အရင် section မှာ ပြောပြခဲ့တဲ့ method နဲ့ very similar ဖြစ်ပါတယ်။ ဒီ method ရဲ့ elements စကားလုံးမှာဆိုရင် 's' ကို ထည့်ပေးရမှာဖြစ်ပါတယ်။ 's' ဆိုတာလေးမပါခဲ့ရင်တော့ သူဟာ အလုပ်လုပ်မှာမဟုတ်ပါဘူး။ နောက် သူဟာ collection of elements ကို returns ပြန်ပေးတာဖြစ်ပါတယ်။ Collection တစ်ခုဟာ array တစ်ခုနဲ့ တူပါတယ်။ ကျွန်တော်ဟာ သူတို့အားလုံးကို access လုပ်ထားတဲ့ collection တစ်ခုလုံးပေါ်မှာသူ့ရဲ့ index or loop ကို အသုံးပြုပြီးတော့ element တစ်ခုကို access လုပ်နိုင်ပါတယ်။

Code Example

        
                const myList = document.getElementsByTagName('li');

                for(let i = 0; i < myList.length; i += 1){
                    myList[i].style.color = 'purple';
                }
