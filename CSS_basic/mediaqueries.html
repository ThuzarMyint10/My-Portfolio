Media Query For Desktop Resolution

ကျွန်တော်တို့ဟာ page တစ်ခုကို low and high fidelity mockups မှာ ဘယ်လို create လုပ်ရတယ်ဆိုတာကို လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခုကျွန်တော်တို့ fluid grids ကို ဘယ်လို media queries ကိုအသုံးပြုပြီးတော့ ဘယ်လို set လုပ်ပြမှာဖြစ်ပါတယ်။ media queries ဟာဆိုရင် responsive web design ရဲ့ last major component ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ ဘယ်လို web page ကိုပဲ create လုပ်လုပ် media queries ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ ဘာလို့လဲဆိုရင် different devices တွေနဲ့ different screen resolutions တွေမှာဆိုရင် ကျွန်တော်တို့ create လုပ်လိုက်တဲ့ web page တစ်ခုဟာ ကြည့်တဲ့အခါမှာ အဆင်ပြေအောင် media queries ကိုအသုံးပြုပေးတာပဲဖြစ်ပါတယ်။အဲ့မှာဆိုရင် 1700px ဟာ larger screen resolution တွေအတွက် ဖြစ်ပြီးတော့ 1400px ဟာဆိုရင် ပုံမှန်ရှိတဲ့ screen resolution တွေအတွက်ပဲဖြစ်ပါတယ်။ 1200px ကတော့ small screen resolution တွေအတွက်ဖြစ်ပါတယ်။ အဲ့မှာကျွန်တော်တို့ apply လုပ်ချင်တဲ့ element တိုင်းကို apply လုပ်နိုင်ပါတယ်။
