CSS Variables Introduction

Sass or Less or PostCSS ကဲ့သို tool တွေလို CSS preprocessor ကိုအသုံးပြုတဲ့အခါမှာ style sheet ထဲမှာ values တွေကို store and keep track လုပ်ဖို့ရန်အတွက် variables တွေကို declare လုပ်ဖို့လိုပါတယ်။ CSS မှာ variables တွေကို special processing tools တွေမလိုဘဲနဲ့ တိုက်ရိုက်အသုံးပြုနိုင်ပါတယ်။ Variables တွေဟာ programming မှာ အရေးကြီးတဲ့ concept တစ်ခုပဲဖြစ်ပါတယ်။ သူတို့ကို information တွေကို store and keep track လုပ်ဖို့ရန်အတွက် နည်းလမ်းတစ်ခုပဲဖြစ်ပါတယ်။ CSS variables တွေကို custom properties လို့လဲခေါ်ပါတယ်။ ဘာကြောင့်လဲဆိုရင် သူတို့ဟာ regular CSS properties နဲ့ similar ဖြစ်ပါတယ် ပြီးတော့ သူဟာ custom property notation လို့ခေါ်တဲ့ special syntax ကိုအသုံးပြုထားတဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ သူ့ကို create လုပ်မယ်ဆိုရင် property name ရဲ့ ရှေ့မှာ two dashes ကို ထည့်ပေးရမှာဖြစ်ပါတယ်။ အဲ့ဒါဟာ browser ကို ဒါဟာ custom property ဖြစ်တယ်ဆိုတာကို instruct လုပ်ပေးလိုက်တာပဲဖြစ်ပါတယ်။ Property name ရဲ့နောက်မှာ value ကို assign လုပ်ပေးရမှာဖြစ်ပါတယ်။ Code ထဲမှာ သူ့ရဲ့ value ကို retrieve လုပ်ဖို့ရန်အတွက် CSS bar function ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ အဲ့ဒါကို technically အရတော့ Custom Properties လို့ခေါ်ပြီးတော့ developers တွေကတော့ CSS variables လို့ခေါ်ကြပါတယ်။

Declarig and Using Variables

CSS variables ကို style sheet ထဲမှာအသုံးပြုမယ်ဆိုရင် steps နှစ်ခုရှိပါတယ်။ First, custom property notation ကိုအသုံးပြုပြီးတော့ selector တစ်ခုထဲမှာ variable ကို declare လုပ်ပေးရမှာဖြစ်ပါတယ်။ ပြီးရင် CSS R function ကိုအသုံးပြုပြီးတော့ variable or custom property ကို reference လုပ်ပေးရမှာဖြစ်ပါတယ်။ CSS variables တွေဟာဆိုရင် property value pairs လိုပဲ အလုပ်လုပ်တာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ property name ကို ကိုယ်ကြိုက်တဲ့ name ပေးပြီးတော့ CSS Property ကို write လုပ်နိုင်ပါတယ်။ ပြီးရင် name ရဲ့ ရှေ့မှာ dashes နှစ်ခုကိုခံပြီးရင် အဲ့ variable တစ်ခုအနေနဲ့ identify လုပ်ပေးလိုက်တာပဲဖြစ်ပါတယ်။ သူ့တိုကို declare လုပ်တဲ့ အချိန်မှာ element ရဲ့ rule set ထဲမှာ declare လုပ်ပေးရမှာဖြစ်ပါတယ်။ သူ့ကို rule ရဲ့အပြင်မှာ declare လုပ်လိုက်မယ်ဆိုရင် သူတို့ဟာ အလုပ်လုပ်မှာမဟုတ်ပါဘူး။ သူ့တို့ကိုလည်း variable တစ်ခုအနေနဲ့ style sheet ထဲမှာ အသုံးပြုလို့မရနိုင်ပါဘူး။ Variables တွေကို declare လုပ်လိုက်တဲ့အခါမှာမှာ brand color, base font size စတဲ့ elements တွေကို reference လုပ်ပေးမှာဖြစ်ပါတယ်။ Example မှာဆိုရင် :root pseudo class ကိုအသုံးပြုပြီးတော့ variables တွေကို declare လုပ်ထားတာပဲဖြစ်ပါတယ်။

CSS variables ကို create လုပ်တာဟာ preprocessor variables နဲ့ equivalent ဖြစ်နေတာပဲဖြစ်ပါတယ်။ သူတို့မှာ တူညီတဲ့ functionality တွေကို offer လုပ်ပေးပါတယ်။ Variables တွေဟာ CSS ထဲမှာ ဘယ်အရာကိုမဆို လုပ်ဆောင်နိုင်ပါတယ်။ Preprocessor variables နဲ့ဆိုရင်တော့ မလုပ်ဆောင်နိုင်ပါဘူး။ Preprocessors တွေဟာ browser ထဲမှာ run အလုပ်နိုင်ပါဘူး။ Variables တွေကို source file တစ်ခုထဲမှာ declare လုပ်လိုက်တဲ့အချိန်မှာ preprocessors တွေဟာ CSS နဲ့ compile လုပ်တဲ့အချိန်မှာ variables နေရာကို သူ့ရဲ့ static values နဲ့ replace လုပ်လိုက်တာပဲဖြစ်ပါတယ်။ CSS variables တွေဟာ code ကို compiled လုပ်တာထက် ပိုပြီးတော့ flexible ဖြစ်ပါတယ်။ ဘာကြောင့်လဲဆိုရင် သူတို့ဟာ browser ထဲမှာရှိတဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ CSS variables ရဲ့ value ကို update လုပ်လိုက်တဲ့အချိန်မှာ browser ဟာ run time မှာ variables ရဲ့ reference လုပ်တဲ့ နေရာတွေကို update လုပ်ပေးမှာဖြစ်ပါတယ်။ နောက် section မှာဆိုရင် CSS variables ဟာ ဘယ်လို cascade လုပ်တယ် သူ့ကိုရဲ့ parent ဆီကနေ value ကို ဘယ်လို inherit လုပ်တယ်ဆိုတာကို ပြောပြပေးမှာဖြစ်ပါတယ်။

ကျွန်တော်တို့ဟာ CSS variables ကို define လုပ်ဖု့ိရန်အတွက် သူရဲ့ scope ကိုစ ပြီးတော့ define လုပ်တာကို လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ Root မှာ variable တစ်ခုကို implies လုပ်တဲ့အခါမှာ variable ဟာ elements အားလုံးအတွက် available ဖြစ်လာတာဖြစ်ပါတယ်။ Root ကို အသုံးပြုတာဟာ global variables တွေကို set လုပ်တဲ့ great way တစ်ခုပဲဖြစ်ပါတယ်။ သူဟာ variables တွေကို တစ်နေရာထဲမှာ keep လုပ်ထားပြီးတော့ maintenance လုပ်ဖို့ရန်လည်း လွယ်ကူပါတယ်။ အမြဲတမ်းတော့ variables တွေကို root ထဲမှာ declare လုပ်ဖို့မလိုပါဘူး။ CSS variables တွေဟာ standard cascade rules ကို follow လုပ်တဲ့ အတွက်ကြောင့် variables တွေဟာ regular CSS properties လို behave လုပ်နိုင်ပါတယ်။ Variable တစ်ခုကို specificity ရဲ့ different levels မှာ define or reset လုပ်နိုင်ပါတယ်။

Code in JavaScript

        
                const body = document.querySelector('body');
                const ball = document.querySelector('.ball');
                
                body.addEventListener('click', e => {
                    ball.style.setProperty('--pos-x', e.clientX - 80);
                    ball.style.setProperty('--pos-y', e.clientY - 80);
                    ball.style.setProperty('--ball--bg', randomHex())
                });
                
                //set a rondom hex value
                function randomHex() {
                    let max = 1 << 24;
                    let hex = (max + Math.floor(Math.random() * max)).toString(16).slice(-6);
                    return `#${hex}`;
                }

How to use CSS Variables

အခုဆိုရင် custom properties တွေဟာ var function မှာ ဘယ်လို အလုပ်လုပ်တယ်ဆိုတာကို သိရှိပြီးဖြစ်ပါတယ်။ First, var ကို selector တစ်ခုအနေနဲ့ အသုံးပြုတဲ့အခါမှာ invalid ဖြစ်နိင်ပါတယ်။ Var ဟာ property name တစ်ခုအနေနဲ့ valid ဖြစ်တာမဟုတ်ပါဘူး။ Var ကို CSS values တွေမှာလည်း အသုံးပြိုနိုင်ပါတယ်။ Var ကို media query expression ထဲမှာလည်းအသုံးပြုလို့မရပါဘူး။ နောက် var function ကို length value ကို create လုပ်တဲ့ခါမှာလည်းအသုံးပြုလို့မရပါဘူး။ Length value ကို build လုပ်ဖို့ဆိုရင် calc ကို အသုံးပြုပြီးမှ var function ကိုအသုံးပြုပေးရမှာဖြစ်ပါတယ်။ CSS variables ဟာဆိုရင် useful and fun feature တွေပဲဖြစ်ပါတယ်။
