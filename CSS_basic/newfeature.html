Calc Method

နောက်ထပ်ပြောပြပေးမှာကတော့ Calc function ပဲဖြစ်ပါတယ်။ Calc CSS method ဟာဆိုရင် mathematical expressions တွေနဲ့ length units အတွက် values ကို calculate လုပ်တဲ့နေရာမှာ အသုံးပြုနိုင်ပါတယ်။ Calc ကို ဘယ်နေရာမှာပဲဖြစ်ဖြစ် pixel or percentage တွေနဲ့ လိုအပ်သလို expressions နဲ့ တွဲပြီးတော့ အသုံးပြုနိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ sizing units ရဲ့ mixture ကို အသုံးးပြုပြီးတော့ HTML elements တွေရဲ့ size ကို specify လုပ်နိုင်ပါတယ်။ Calc ကိုအသုံးပြုလိုက်တဲ့အခါမှာ browser ဟာဆိုရင် correct sizes ကို output ထုပ်ပေးမှာဖြစ်ပါတယ်။ Mix percentages and absolute values and mixing sizing units တွေဟာ layouts ကို ပိုပြီးတော့ flexible ဖြစ်စေပါတယ်။

Code in CSS

        
                html {
                    height: 100%;
                }
                
                body {
                    margin: 0;
                    height: 100%;
                    color: #fff;
                }
                
                .wrapper {
                    background: #1e90ff;
                    height: -webkit-calc(80% - 100px);
                    /* width: -webkit-calc(50% - 120px); */
                    width: -webkit-calc(100%/3 + 2em);
                }

FlexBox Property

နောက်ထပ်ကတော့ Flexible layout or Flex box feature ပဲဖြစ်ပါတယ်။ Floats ဟာဆိုရင် common layout method တစ်ခုဖြစ်ပါတယ်။ သူဟာဆိုရင် complicates box model တစ်ခုဖြစ်ပါတယ်။ ဘာကြောင့်လဲဆိုတော့ သူဟာ elements တွေကို normal document flow မှာ နေရာယူပေးတဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ New flex box feature ဟာဆိုရင် powerful ဖြစ်ပါတယ်။ သူဟာဆိုရင် layout အတွက် CSS ကို more efficient ဖြစ်အောင်လုပ်ဆောင်နိုင်ပါတယ်။ Layout issues တွေကိုလည်း solve လုပ်ပေးပါတယ်။ Flex box ဟာဆိုရင် 1 dimensional CSS layout ကို handle လုပ်တာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ elements ကို directions အမျိုးမျိုးနဲ့ lay out လုပ်နိုင်ပါတယ်။ Boxes grow, shrink, and autofill space တွေကိုလည်း လုပ်ဆောင်နိုင်ပါတယ်။ Flex box ရဲ့ default laid out ဟာဆိုရင် column ပဲဖြစ်ပါတယ်။ Flex box ကိုအသုံးပြုမယ်ဆိုရင် flex formatting context ကို establish လုပ်ဖို့လိုပါတယ်။ Flex box ကို အသုံးပြုဖို့ဆိုရင် display property ကို add လုပ်ပေးဖို့လိုပါတယ်။သူ့နောက်မှာ value ကို webkit vendor prefix နဲ့ follow by flex ဆိုပြီးတော့ ရေးပေးရမှာဖြစ်ပါတယ်။

Multi-column Property

အခုလေ့လာမှာကတော့ layout feature ဖြစ်တဲ့ multi-column layout အကြောင်းပဲဖြစ်ပါတယ်။ Multi-column layout ဟာဆိုရင် flex box ကို resemble လုပ်တာဖြစ်ပြီးတော့ layout flexibility ကိုလည်း ပေးပါတယ်။ သူ့ကို multiple text columns တွေကို create လုပ်တဲ့ခါမှာအသုံးပြုနိုင်ပါတယ်။ သူဟာ new section တစ်ခုကိုတော့ create လုပ်ပေးတာမဟုတ်ပါဘူး။ သူဟာ same box ထဲမှာ box's content တွေကို multiple columns တွေနဲ့ flow ခြင်းဖြင့် block layout mode ကို extends လုပ်ပေးတာပဲဖြစ်ပါတယ်။ နောက်ထပ် သူဟာ column widths, gaps, and rules တို့ကိုလည်း determine လုပ်ပေးနိုင်ပါတယ်။ Multi-column layout ဟာာ box container ရဲ့ new type of column ကို introduces ပဲဖြစ်ပါတယ်။ Example မှာဆိုရင် column count property ကိုအသုံးပြုတဲ့ အခါမှာ number of columns ကို set လုပ်ပေးရမှာဖြစ်ပါတယ်။ Column rule property ဟာ border property နဲ့ အမျိုးအစားတူတူ ပဲဖြစ်ပါတယ်။ နောက်ထပ် cloumn span property တစ်ခုကတော့ element တစ်ခုအတွက် several columns ကို across လုပ်ပြီးတော့ span အနေနဲ့ ဖြစ်အောင်ပြုလုပ်ပေးနိုင်ပါတယ်။

Code in CSS

        
                body{
                    font: 100%/1.5 sans-serif;
                    padding: 8% 5%;
                }
                div {
                    -webkit-column-count: 4;
                    -webkit-column-gap: 3em;
                    -webkit-column-rule: 1px dotted;
                    -webkit-column-width: 250px;
                }
                h2 {
                    -webkit-column-span: all;
                }
