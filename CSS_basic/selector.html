Introduction to CSS Selector

ဒီအပိုင်းမှာတော့ basic selector concepts အကြောင်းကို လေ့လာသွားမှာဖြစ်ပါတယ်။ အရင် ကျွန်တော်တို့ဟာ base selectors တွေဖြစ်တဲ့ type, ID, class and descendant selectors အကြောင်းတွေကို လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခု selector ရဲ့ new batch တစ်ခုလုံးကို လေ့လာကြမှာဖြစ်ပါတယ်။ Selectors တွေဟာ CSS ရဲ့ most important and powerful part ပဲဖြစ်ပါတယ်။ ဘာကြောင့်လဲဆိုတော့ သူတို့ဟာ HTML ရဲ့ elements ကို target လုပ်ပြီးတော့ style တွေကို သူတို့ကို apply လုပ်ဖို့ allow လုပ်ပေးတဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ အဲ့ common ways တွေအပြင် ကျွန်တော်တို့ဟာ type, ID and class selectors တွေနဲ့ target လုပ်တဲ့အကြောင်းကိုလည်း လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခု ဒီသင်ခန်းစာမှာဆိုရင် attributes selectors, child selectors, selector combinations and helpful ဖြစ်တဲ့ pseudo-classes တွေအကြောင်းကို လေ့လာမှာဖြစ်ပါတယ်။ နောက်ထပ် ဘယ်လို advanced selectors တွေဟာ simple expression တွေမှာ elements တွေရဲ့ combination ကို target လုပ်ဖို့ရန်အတွက် အသုံးပြုနိုင်တယ်ဆိုတာကို တွေ့ရမှာဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် CSS ကနေ page ထဲကို virtual elements and content တွေကို insert လုပ်ဖို့ရန်အတွက် pseudo-elements လို့ခေါ်တဲ့ special selectors တွေကိုလည်းအသုံးပြုနိုင်ပါတယ်။

Class and Div Selector

HTML elements ဟာဆိုရင် attributes တွေရဲ့ various types တွေကို accepts လုပ်နိုင်ပါတယ်။ CSS ဟာဆိုရင် element တစ်ခုရဲ့ ဘယ် attributes နဲ့ values တွေကိုမဆို target လုပ်ပြီးတော့ element ကို style တွေ apply လုပ်နိုင်ပါတယ်။ example page ရဲ့ element တစ်ခုချင်းစီမှာဆိုရင် သူတို့ရဲ့ opening tags တွေထဲမှာ attributes တွေရှိကြပါတယ်။ ဥပမာ type attribute, for attribute, placeholder attribute တို့ဖြစ်ပါတယ်။ Attributes selectors ကိုအသုံးပြုပြီးတော့ elements တွေကို သူတို့ရဲ့ attributes or သူတို့ရဲ့ values တွေကိုအခြေခံပြီးတော့ target လုပ်နိုင်ပါတယ်။ Attribute selector တစ်ခုကို CSS နဲ့ define လုပ်လိုက်တဲ့အခါမှာ သူဟာ element ရဲ့ class or ID name တွေလို matching လုပ်စရာမလိုပါဘူး သူဟာ သူကိုယ်တိုင် match ဖြစ်နေပြီးသားဖြစ်ပါတယ်။ ဥပမာ အခြေခံအကျဆုံးဖြစ်တဲ့ attribute selector ဟာ element attribute တစ်ခုကိုပဲ target လုပ်ပေးပါလိမ့်မယ်။ class attribute နဲ့ target လုပ်ပြီးတော့ page ရဲ့ elements အားလုံးကို target လုပ်ချင်တယ်ဆိုရင် []=square brackets ကို create လုပ်ပြီးတော့ သူ့ထဲမှာ class ဆိုပြီးတော့ ရေးပေးရမှာဖြစ်ပါတယ်။ အဲ့နောက်မှာ {}= curly braces လိုက်ပြီးတော့ အဲ့ထဲမှာ CSS ရဲ့ ကိုယ် apply လုပ်ချင်တဲ့ property တွေကို ရေးပေးရမှာဖြစ်ပါတယ်။ page မှာဆိုရင် two form elements နှစ်ခုမှာ class attribute တစ်ခုစီရှိကြပါတယ်။ class selector ထဲမှာ specific class value တစ်ခုကို ပေးပြီးတော့ apply လုပ်နိုင်ပါတယ်။ ဥပမာ [class="form_contact"] ဆိုပြီးတော့ create လုပ်နိုင်ပါတယ်။ သူကတော့ ကိုယ် apply လုပ်ချင်တဲ့ class value ကို set of quotes ထဲမှာရေးပေးရတာပဲဖြစ်ပါတယ်။ နောက်ထပ် ကျွန်တော်တို့ဟာ div ကို target လုပ်နိုင်ပါတယ်။ page ရဲ့ id container value ကို target လုပ်မှာဖြစ်ပါတယ်။

Pseudo Classes

အခုလေ့လာမှာကတော့ CSS pseudo classes တွေအကြောင်းပဲဖြစ်ပါတယ်။ CSS မှာဆိုရင် helpful ဖြစ်တဲ့ အခြား pseudo classes တွေရှိပါတယ်။ သူတို့ဟာဆိုရင် elements တွေရဲ့ page ပေါ်မှာရှိတဲ့ position နဲ့ elements တွေရဲ့ အခြား elements တွေနဲ့ relation ပေါ်မှာမူတည်ပြီးတော့ target လုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူတို့ဟာတော့ CSS selectors advanced ရဲ့နောက်မှာရှိတဲ့ real power and flexibility တွေပဲဖြစ်ပါတယ်။ Pseudo classes တွေဟာဆိုရင် elements တွေရဲ့ first and last child ကိုလည်း target လုပ်နိုင်ပါတယ်။ အခု first child structural pseudo classes ကိုအသုံးပြုတဲ့အခါမှာ browser ဟာ first element ကိုအလုပ်လုပ်ပေးသွားမှာဖြစ်ပါတယ်။ last child structural pseudo classes ကိုအသုံးပြုမယ်ဆိုရင် browser ဟာ last element ကိုအလုပ်လုပ်ပေးမှာဖြစ်ပါတယ်။ Structural pseudo classes ကို create လုပ်တဲ့အခါမှာ first child အတွက်ဆိုရင် အခု example မှာ ကျွန်တော်တို့ target လုပ်ချင်တဲ့ element ဟာ li tag ထဲမှာရှိတဲ့အတွက်ကြောင့် li:first-child ဆိုပြီးတော့ ကြေငြာပေးထားပဲဖြစ်ပါတယ်။ last child အတွက်ဆိုရင် first-child နေရာမှာ last-child ကိုပြောင်းပေးရမှာပါ။

Browser ဟာဆိုရင် only child elements, and empty elements တွေကိုလည်း target လုပ်နိုင်ပါတယ်။ Only child element ဟာဆိုရင် parent element ရဲ့ child ကိုပဲ target လုပ်ပေးမှာဖြစ်ပါတယ်။ only child selector ကို create လုပ်ချင်တယ်ဆိုရင် full colon ရေးပြီးတော့ အဲ့နောက်မှာ only-child ကိုရေးပေးရမှာဖြစ်ပါတယ်။ Page ထဲမှာဆိုရင် child ဟာဆိုရင် li တွေပဲဖြစ်ပါတယ်။ Body ထဲမှာနောက်ထပ် h1 ကို ထပ်ထည့်လိုက်တဲ့အခါမှာ အဲ့ H1 လေးဟာ ul element နဲ့ sibling ဖြစ်သွားတာပဲဖြစ်ပါတယ်။ Empty ဆိုတာကတော့ empty content ကို ြပောတာပဲဖြစ်ပါတယ်။ empty ကို target လုပ်တဲ့အခါမှာတော့ :empty ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။

            }
            li:empty{
                background-color: tomato;

Substring Matching Attribute Selectors

ကျွန်တော်တို့ဟာ attribute selectors တွေနဲ့ elements တွေကို target လုပ်ခဲ့ပြီးဖြစ်ပါတယ်။ ဒီတစ်ခါမှာတော့ substring matching attribute selectors လု့ိခေါ်တဲ့ powerful ဖြစ်တဲ့ selectors 3 ခုကို အသုံးပြုမှာဖြစ်ပါတယ်။ အဲ့ဒါတွေဟာဆိုရင် attribute တစ်ခုရဲ့ value မှာရှိတဲ့ specific pieces တွေကို target လုပ်ပေးတာပဲဖြစ်ပါတယ်။ အရင် ကျွန်တော်တို့ဟာ element တစ်ခုရဲ့ any attribute and value တွေကို target လုပ်တဲ့အကြောင်းကိုလေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခု ကျွန်တော်တို့ဟာ letters or words တွေလို တိကျတဲ့ attribute ရဲ့ pieces တွေကို substring matching attribute selectors တွေကိုအသုံးပြုမှာဖြစ်ပါတယ်။ pieces တွေကို substrings လို့ခေါ်ပါတယ်။ ဥပမာ example page ထဲမှာဆိုရင် a tag တွေရှိပါတယ်။ အောက်က a tag တွေဟာ http:// နဲ့ဟာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ attributes မှာပေးထားတဲ့ values ရဲ့ start ကိုလည်း target လုပ်ပြီးတော့ CSS တွေ apply လုပ်နိုင်သလို values ရဲ့ end ကိုလည်း target လုပ်ပြီးတော့ CSS တွေကို apply လုပ်နိုင်ပါတယ်။ Values ရဲ့ start ကို target လုပ်တဲ့အခါမှာ ဥပမာ a[herf^="http://"] ဆိုပြီးတော့ရေးပေးရမှာဖြစ်ပါတယ်။ Carat character(^) လေးဟာ browser ကို ဒါဟာ code ရဲ့ specific piece လေးဖြစ်ပါတယ်ဆိုတာကို ပြောပြပေးတာပဲဖြစ်ပါတယ်။ Values ရဲ့ start ကို target လုပ်သလိုပဲ end ကို target လုပ်တဲ့နေရာမှာလည်း a[herf$=".pdf"] ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။ အဲ့မှာဆိုရင်လည်း "$" sign လေးဟာဆိုရင် browser ကို attribute ရဲ့ value ရဲ့ end substring ဖြစ်ပါတယ်ဆိုတာကို ပြောပြပေးတာပဲဖြစ်ပါတယ်။

Third Substring Matching Attribute Selectors

Third substring matching attribute selector အကြောင်းကိုပြောပြပေးမှာဖြစ်ပါတယ်။ သူဟာဆိုရင် attribute ရဲ့ values မှာ ကြေငြာထားတဲ့ code ရဲ့ piece တွေကို target လုပ်နိုင်ပါတယ်။ အရင် ကျွန်တော်တို့ဟာ begins and end တွေကို target လုပ်ခဲ့ပြီးဖြစ်ပါတယ်။ အခုတစ်ခါမှာတော့ third ဖြစ်တဲ့ contains selector ကိုအသုံးပြုမှာဖြစ်ပါတယ်။ Example page မှာရှိတဲ့ a tag ထဲမှာ values မှာကြေငြာထားတာကတော့ 3 ပိုင်းရှိပါတယ်။ အခု အလယ်မှာရှိတဲ့ download ဆိုတဲ့ code ကို target လုပ်မှာဖြစ်ပါတယ်။ သူ့ကို create လုပ်တဲ့အခါမှာဆိုရင် a[href*="download"] ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။ "*" ဟာဆိုရင် browser ကို href attribute မှာ "download" ဆိုတဲ့ word ပါတဲ့ anchor element တိုင်းကို selecting လုပ်မယ်လို့ပြောလိုက်တာပဲဖြစ်ပါတယ်။ နောက်ထပ် img elements တွေမှာဆိုရင် thumbails ဆိုတဲ့ word ကို target လုပ်နိုင်ပါတယ်။ thumbails ကို ရေးတဲ့အခါမှာလည်း ကျွန်တော်တို့ဟာ စကားကိုအပြည့်အစုံ မရေးလည်းရပါတယ်။ အခုကျွန်တော်ပြောပြခဲ့တဲ့ substring matching attribute selectors ဟာဆိုရင် ပိုပြီးတော့ powerful ဖြစ်ပါတယ်။ ဘာလို့လဲဆိုရင် သူတို့ဟာ attribute တစ်ခုရဲ့ ဘယ်အပိုင်းကိုမဆို target လုပ်နိုင်ပြီးတော့ ရေးတဲ့အခါမှာလည်း full value ကိုပေးစရာမလိုတဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။

User Action Pseudo-classes

ဒီသင်ခန်းစာမှာတော့ ကျွန်တော်တို့ဟာ user action pseudo-classes အကြောင်းကိုပြောပြပေးသွားဖြစ်ပါတယ်။ သူဟာဆိုရင် element တစ်ခုနဲ့ user's interaction ပေါ်မှာအခြေခံပြီးတော့ elements တွေကို target လုပ်ပေးတာပဲဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် focus pseudo class ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ သူ့ကို create လုပ်တဲ့ အခါမှာဆိုရင် ကျွန်တော်တို့ target လုပ်ချင်တဲ့ element ကို ရှေ့ကရေးပြီးတော့ နောက်က ":" လိုက်ပေးရပါမယ်။ ပြီးမှာ focus pseudo class ဖြစ်တဲ့ focus ကိုရေးပေးရမှာဖြစ်ပါတယ်။ နောက်ထပ် ကျွန်တော်တို့ဟာ disabled ဖြစ်နေတဲ့ elements တွေကိုလည်း target လုပ်နိုင်ပါတယ်။ target လုပ်တဲ့နေရာမှာ ":disabled" ဆိုပြီးတော့ create လုပ်ပေးရမှာမယ်။ Disable ဆိုတဲ့ value ကို ကျွန်တော်တို့ဟာ HTML file ထဲမှာ disabled ဖြစ်ချင်တဲ့ element မှာထည့်ပေးခဲ့ရမှာဖြစ်ပါတယ်။ နောက်ထပ် check pseudo class ပဲဖြစ်ပါတယ်။Elements တွေဟာ check state ကိုရောက်သွားတဲ့ အချိန်မှာ changes တွေဖြစ်သွားတာမျိုးကိုပြောချင်တာပါ။ ဥပမာ radio button or check box တို့ကို click လုပ်လိုက်တဲ့အခါမှာ သူ့တို့ဘေးမှာ ရှိတဲ့ text လေးတွေဟာ bold ဖြစ်သွားတာတို့ကိုလုပ်ဆောင်နိုင်ပါတယ်။

Advanced Selectors

အစောပိုင်းမှာဆိုရင် first and last child တွေကို structural pseudo-classes တွေနဲ့ ဘယ်လို target လုပ်ရတယ်ဆိုတာကို လေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခုလေ့လာမှာကတော့ nth-child pseudo-classes တွေဟာ child elements ရဲ့ any combination ကိုမဆို ဘယ်လို target လုပ်နိုင်တယ်ဆိုတာကိုလေ့လာသွားမှာဖြစ်ပါတယ်။ Nth-child ဟာဆိုရင် useful and powerful ဖြစ်တဲ့ structural pseudo-class တစ်ခုပဲဖြစ်ပါတယ်။ ဘာကြောင့်လဲဆိုရင် element တစ်ခုဟာ parent element ရဲ့ ဘယ်နေရာမှာဘဲရှိရှိ target လုပ်နိုင်တဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ First and last child pseudo-classes တွေနဲ့ မတူတာကတော့ သူဟာဆိုရင် child element ရဲ့ any child or any combination တွေကို target လုပ်နိင်ပါတယ်။ Example page မှာဆိုရင် ul နဲ့ရေးထားတဲ့ li တွေရှိပါတယ်။

ကျွန်တော်တို့ဟာ nth-child selector ကိုအသုံးပြုမယ်ဆိုရင် li:nth-child() ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။ Parentheses ထဲမှာဆိုရင် keyword or odd or even ကို ကြိုက်တာ ပေးလို့ရပါတယ်။ Odd ပေးလိုက်တဲ့အခါမှာဆိုရင် ကျွန်တော်တို့ရဲ့ list ထဲမှာရှိတဲ့ odd အကြိမ်မြောက် items တွေကိုပဲ CSS ရဲ့ style တွေကို apply လုပ်ပေးမှာဖြစ်ပါတယ်။ Even ဆိုရင် list ရဲ့ even ကြိမ်မြောက် items တွေကို apply လုပ်ပေးမှာဖြစ်ပါတယ်။ Key words တွေပေးတဲ့အခါမှာဆိုရင် 1,2,3 or 4 ဆိုပြီးတော့ ကျွန်တော်တို့ apply လုပ်ချင်တဲ့ item ရဲ့ position ကို ပေးနိုင်ပါတယ်။ အဲ့လိုပေးချင်တဲ့အခါမှာ ကျွန်တော်တို့ဟာ တစ်ခုကျော်၊ သုံးခုကျော်၊ လေးခုကျော် ပေးချင်တဲ့အခါမှာဆိုရင် nth-child(an+b) ဆိုပြီးတော့ parentheses ထဲမှာရှိတဲ့ b ဟာဆိုရင်ကျွန်တော်တို့ စပြီးတော့ apply လုပ်ချင်တဲ့ child element ရဲ့ position ပဲဖြစ်ပါတယ်။ a နေရာမှာ 2 လို့ထည့်လိုက်တယ်ဆိုပါတော့ အစကနေပြီးတော့ နောက်ထပ် 2 ကြိမ်မြောက် child element ကို apply လုပ်ပေးပါလို့ browser ကိုပြောလိုက်တာပဲဖြစ်ပါတယ်။ နောက်က n လေးကတာ့ counter လို့ခေါ်ပါတယ်။ ကျွန်တော်တို့ဟာ parentheses ထဲမှာဆိုရင် negative sign တွေနဲ့လဲပေးလို့ရပါတယ်။ ဥပမာ nth-child(-3n) ဆိုရင် သူဟာအပေါ်ဆုံး 3 ခုကို apply လုပ်ပေးမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ 0 ပေးလိုက်တဲ့အခါမှာဆိုရင် အလုပ်လုပ်မှာမဟုတ်ပါဘူး။ ပြီးတော့ parentheses ထဲမှာ 3n+3 ဆိုရင်လည်း အစကနေ 3 ကြိမ်မြောက် items တွေကိုပဲ apply လုပ်ပေးမှာဖြစ်ပါတယ်။

The Nth of Type Pseudo-classes

ဒီတစ်ခါမှာတော့ nth structural pseudo classes တွေကိုအသုံးပြုပြီးတော့ elements တွေရဲ့ particular types တွေကို target လုပ်မှာဖြစ်ပါတယ်။ Nth pseudo class ဟာဆိုရင် element တစ်ခုကို target လုပ်တဲ့နေရာမှာ element ရဲ့ position ကို အေခြခံပြီးတော့ target လုပ်နိုင်တဲ့အပြင် element ရဲ့ specific type ကိုအခြေခံပြီးတော့လည်း target လုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ element တစ်ခုရဲ့ type ပေါ်မှာ ခြေခံပြီးတော့ target လုပ်ချင်တဲ့အခါမှာဆိုရင် "nth-of-type()" ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။ Parentheses ထဲမှာဆိုရင် ကျွန်တော်တို့ target လုပ်ချင်တဲ့ element ရဲ့ position ကို key word နဲ့ ပေးနိုင်ပါတယ်။ Example page ရဲ့ div ထဲမှာဆိုရင် head တွေရှိပါတယ်။ div:nth-of-type(4) လို့ပေးလိုက်တဲ့အခါမှာ သူဟာ head တွေဘယ်နေရာမှာဘဲရှိရှိ head တွေကို ကျော်ပြီးတော့ count လုပ်ပြီး 4 ခုမြောက် item ကို apply လုပ်ပေးမှာဖြစ်ပါတယ်။ နောက်ဆုံးတစ်ခုကိုပဲ apply လုပ်ချင်ရင်တော့ "nth-last-of-type" ဆိုပြီးတော့ create လုပ်ပေးရမှာဖြစ်ပါတယ်။

Code in CSS

        
                /* div:nth-of-type(3){
                    background: #52bab3;
                    color: white;
                } */
                div:nth-last-of-type(3){
                    background: #52bab3;
                    color: white;
                }

Negation Pseudo-classes

အခု လေ့လာမှာကတော့ negation pseudo class အကြောင်းပဲဖြစ်ပါတယ်။ သူဟာဆိုရင် element တစ်ခုကို ကျွန်တော်တို့ apply မလုပ်စေချင်တဲ့ အခါမှာအသုံးပြုနိင်ပါတယ်။ သူ့ကို not selector လု့ိခေါ်ပါတယ်။ သူ့ကို create လုပ်တဲ့အခါမှာ ကျွန်တော်တို့ apply လုပ်ချင်တဲ့ element ကို ရှေ့ရေးပြီးတော့ :not() ကိုရေးပေးရမှာဖြစ်ပါတယ်။

Code in CSS

        
            /*********************Negation Pseudo-classes***********/
            input:not([type="submit"])
            {
                box-shadow: inset 0 2px 0 rgba(0, 0, 0, .15);
            }
            .col:not(:first-child),
            nav a:not(:first-child){
                margin-left: 15px;
            }

Pseudo-elements တွေဟာဆိုရင် pseudo-classes တွေနဲ့ similar ဖြစ်ပါတယ်။ Pseudo-class နဲ့ မတူတာကတော့ pseudo-elements တွေဟာ element တစ်ခုရဲ့ တိကျတဲ့ အပိုင်းကို style တွေကို apply လုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူ့မှာဆိုရင် elements and content CSS ကနေ page ထဲကို insert လုပ်နိုင်ပါတယ်။ CSS မှာဆိုရင် pseudo-element ဟာ types 4 ခုရှိပါတယ်။ အခုကျွန်တော်ပြောပြပေးမှာကတော့ first line and first letter pseudo-elements အကြောင်းပဲဖြစ်ပါတယ်။ Page မှာရှိတဲ့ paragraph ရဲ့ first line ကို ကျွန်တော်တို့ apply လုပ်ချင်တဲ့အခါမှာ .intro ဆိုတဲ့ class ကိုခေါ်ပေးရမှာဖြစ်ပါတယ်။ ပြီးတော့ ကျွန်တော်တို့ဟာ paragraph ရဲ့ first line ကိုပဲ apply လုပ်ချင်တာဖြစ်တဲ့အတွက် ":" ခံပြီးတော့ first-line ကိုထည့်ပေးရမှာဖြစ်ပါတ်ယ်။ Pseudo-element နဲ့ pseudo-class နဲ့ မတူတာကတော့ pseudo-classes တွေကို ရေးတဲ့အခါမှာ ":" ဟာ တစ်ခုပဲခံရတာဖြစ်ပြီးတော့ pseudo-elements တွေကို ရေးတဲ့အခါမှာတော့ "::" နှစ်ကိုခံပေးရမှာဖြစ်ပါတယ်။ နောက်ထပ်ကျွန်တော်တို့ဟာ first line ရဲ့ first letter ကို apply လုပ်ချင်တဲ့အခါမှာ ခေါ်ရတဲ့ class ကတော့ .intro ပဲဖြစ်ပါတယ်။ သူ့ရဲ့နောက်ကတော့ first-letter ဆိုပြီးတော့ ရေးပေးရမှာဖြစ်ပါတယ်။

Before and After Pseudo-elements

အခုလေ့လာမှာကတော့ before and after pseudo-elements အကြောင်းပဲဖြစ်ပါတယ်။ သူဟာဆိုရင် element ရဲ့ content နောက်မှာ virtual element တွေကို insert လုပ်နိုင်ပါတယ်။ Before and after ဟာဆိုရင် element တစ်ခုကို extra two layers နဲ့ အလုပ်လုပ်တဲ့အခါမှာ အသုံးပြုနိုင်ပါတယ်။ Before and after ကို generated content လို့ခေါ်ပါတယ်။ သူတို့ကို create လုပ်တဲ့နေရာမှာ ရှေ့က ကျွန်တော်တို့ apply လုပ်ချင်တဲ့ element ကို target လုပ်ပြီးတော့ "h1::before" ဆိုပြီးတော့ ရေးပေးရမှာဖြစ်ပါတယ်။ သူ့ထဲမှာ အသုံးပြုရတဲ့ property ကတော့ content property ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ photo မဟုတ်တဲ့ text တွေပေါ်ချင်တဲ့အခါမှာ ရေးချင်တဲ့ text ကို "" ထဲမှာ ထည့်ရေးပေးရမှာဖြစ်ပါတယ်။ Image တွေဆိုရင်တော့ url နဲ့ ခေါ်ပြီးထည့်ပေးလို့ရပါတယ်။ "" ထဲကို ထည့်ရေးလို့မရပါဘူး။

Attribute Pseudo-classes

အခြား useful ဖြစ်တဲ့ pseudo elements တစ်ခုရှိပါသေးတယ်။ သူတော့ element ရဲ့ content ပေါ်မှာအခြေခံပြီးတော့ content တွေကို ထုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူဟာဆိုရင် element ရဲ့ attribute မှာ ရှိတဲ့ values ကို generate လုပ်ပေးတာဖြစ်ပါတယ်။ သူမှာ content ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ သူ့ထဲမှာ attribute name ကို"attr()" ထည့်ပေးရမှာဖြစ်ပါတယ်။ Pseudo-classes တွေမှာဆိုရင် ကျွန်တော်တို့ဟာ Nth child pseudo class အကြောင်းကို ပထမဆုံးလေ့လာခဲ့တာဖြစ်ပါတယ်။ သူဟာဆိုရင် child elements တွေရဲ့ combination ကို target လုပ်ပေးတာပဲဖြစ်ပါတယ်။ values အနေနဲ့ကတော့ odd or even or keyword တွေကိုပေးနိုင်ပါတယ်။ နောက် nth-of-type ဆိုတဲ့ pseudo class ကိုလေ့လာခဲ့တာဖြစ်ပါတယ်။ သူဟာ parent element ထဲမှာရှိတဲ့ child element ရဲ့ position ပေါ်မှာအခြေခံပြီးတော့ target လုပ်ပေးတာပဲဖြစ်ပါတယ်။ နောက် root pseudo class ပဲဖြစ်ပါတယ်။ သူကတော့ html element အတွက်ပဲ အလုပ်လုပ်ပေးတာဖြစ်ပါတယ်။ နောက်ဆုံးအနေနဲ့ကတော့ pseudo elements အကြောင်းကို လေ့လာ ခဲ့တာပဲဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် virtual elements တွေကို target လုပ်နိင်ပြီးတော့ content တွေကို generate လုပ်ပေးနိုင်ပါတယ်။
