Introduction to Flexbox Laytout

ဒီတစ်ခါ လေ့လာမှာကတော့ webpages တွေကို laying out လုပ်ဖို့ရန်အတွက် အခြေခံအကျဆုံး methods တစ်ခုဖြစ်တဲ့ CSS Flexbox layout ပဲဖြစ်ပါတယ်။ floats, in inline block and absolute positioning တွေလို CSS lab methods တွေမှာဆိုရင် certain limitations တွေရှိပါတယ်။ သူတို့ကို ယနေ့ခောတ် complex responsive websites တွေရဲ့ layout demands ကို handle လုပ်ဖို့ရန်အတွက် designed လုပ်ထားတာတော့ မဟုတ်ပါဘူး။ Floats ဟာဆိုရင် page layout အတွက် intended လုပ်ပေးတာတော့မဟုတ်ပါဘူး။ သူဟာ around images မှာရှိတဲ့ text တွေကို wrapping လုပ်ပေးတာပဲဖြစ်ပါတယ်။ CSS layout basics ကိုလေ့လာခဲ့သလိုပဲ floats မှာ သူရဲ့ တစ်မူထူးခြားတဲ့ behavior ကြောင့် problem လေးတွေရှိနိုင်ပါတယ်။ Flexbox or Flexible box layout ဟာဆိုရင် CSS property ရဲ့ set တစ်ခုပဲဖြစ်ပါတယ်။ သူဟာ page content တွေကို layout လုပ်ဖု့ိရန်အတွက် flexible way တစ်ခုကို prove လုပ်ပေးတာပဲဖြစ်ပါတယ်။ အဲ့ဒါဟာ remarkable layout feature တစ်ခုဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ Flexbox ကိုအသုံးပြုပြီးတော့ navigation links, content columns and images တွေရဲ့ direction, size and order or elements တွေကို change လုပ်နိုင်ပါတယ်။ သူတို့ရဲ့ original size and order တွေကို HTML ထဲမှာ regardless လုပ်ပေးတာပဲဖြစ်ပါတယ်။

Flexbox ကိုအသုံးမပြုခင်မှာ ကျွန်တော်တို့ဟာ flexbox ရဲ့ နောက်မှာရှိတဲ့ terminology and concepts တွေကို နားလည်ထားသင့်ပါတယ်။ Flexbox layout မှာရှိတဲ့ အရေးကြီးဆုံး elements နှစ်ခုကတော့ flex containers and flex items တွေပဲဖြစ်ပါတယ်။ First step အနေနဲ့ flexbox ကို အသုံးပြုမယ်ဆိုရင် flex container ကို create လုပ်ရပါမယ်။ flex container ဟာဆိုရင် flexbox layout ရဲ့ context ကို sets လုပ်ပေးတာပဲဖြစ်ပါတယ်။ flexbox context မှာဆိုရင် flex items တွေပါပါတယ်။ Flex container ထဲမှာ div, ul, header, span element တွေကဲ့သို့သော် block-level or inline element တွေကို ထည့်နိုင်ပါတယ်။ အဲ့ဒါတွေကို flex item လို့ခေါ်ပါတယ်။ ကျွန်တော်တို့ဟာ flexbox ရဲ့ powerful ဖြစ်တဲ့ alignment properties တွေကို အသုံးပြုနိုင်ပါတယ်။ Flexbox ဘယ်လိုအလုပ်လုပ်တယ်ဆိုတာကို နားလည်ဖို့ရန်အတွက် ကျွန်တော်တို့ဟာ flex items တွေကို place လုပ်နိုင်တဲ့ browser ရဲ့ axis ကိုနားလည်ထားဖို့လိုပါတယ်။ Flexbox ဟာဆိုရင် flex items တွေရဲ့ layout direction ကို determine လုပ်ပေးတဲ့ axis နှစ်ခုကို follows လုပ်ပေးတာပဲဖြစ်ပါတယ်။ Items တွေဟာ container ထဲမှာဆိုရင် main axis or cross axis မှာ နေရာယူတာဖြစ်ပါတယ်။ Flexbox ထဲမှာရှိတဲ့ အရာတွေအားလုံးဟာ အဲ့ axis နှစ်ခုနဲ့ relative ဖြစ်နေပါတယ်။ main axis ဟာဆိုရင် primary axis ဖြစ်ပြီးတော့ flex items တွေဟာ သူ့တစ်လျောက်လုံးမှာနေရာယူပါတယ်။ သူဟာ flex container ထဲမှာရှိတဲ့ flex items တွေရဲ့ direction ကို define လုပ်ပေးပါတယ်။ cross axis ဟာ main axis ရဲ့ perpendicular မှာ runs လုပ်ပါတယ်။ axis တစ်ခုချင်းစီမှာဆိုရင် start side and end side ဆိုပြီးတော့ sides နှစ်ခုရှိပါတယ်။ main axis ရဲ့ default ဖြစ်တဲ့ main start and main end တို့ရဲ့ direction ကို left to right သွားတာပြီးတော့ cross ဟာတော့ top to bottom သွားတာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ directions တွေကို အလွယ်တကူ change လုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ main axis တစ်လျောက်မှာရှိတဲ့ horizontal column layout ကို flexbox properties ကို သုံးပြုပြီးတော့ vertical layout ကို switch လုပ်နိုင်ပါတယ်။

Flexbox Layout

အခုကျွန်တော်တို့မှာ basic concepts and terminology အကြောင်းကို သိပြီးခဲ့ပြီးဖြစ်ပါတယ်။ အခုကျွန်တော်တို့ဟာ flexbox ကို အသုံးပြုမှာဖြစ်ပါတယ်။ first step အနေနဲ့ flex box layout ကို create လုပ်ရမှာဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် container ဆိုတဲ့ class ကို flexbox layout ကိုအသုံးပြုမှာဖြစ်ပါတယ်။ flexbox ကို အသုံးပြုတဲ့အခါမှာ display property ကိုအသုံးပြုမှာဖြစ်ပါတယ်။ သူ့မှာဆိုရင် value ကို flex နဲ့ inline-flex ဆိုပြီးတော့ပေးနိုင်ပါတယ်။

Code in HTML 

         
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <meta http-equiv="X-UA-Compatible" content="ie=edge">
                <link rel="stylesheet" href="flexbox.css">
                <title>Flexbox Layout</title>
            </head>
            <body>
                 <div class="container">
                     <div class="item-1 item">Item1</div>
                     <div class="item-2 item">Item2</div>
                     <div class="item-3 item">Item3</div>
                     <div class="item-4 item">Item4</div>
                     <div class="item-5 item">Item5</div>
                     <div class="item-6 item">Item6</div>
                 </div>
            </body>
            </html>

            .container{
                padding: 10px;

            }
            body{

                font-family: Verdana, Geneva, Tahoma, sans-serif;
                color: #fff;

                padding-right: 5%;
                background: #e8e9e9; 
            }
            .item{
                color: #fff;
                padding: 15px;
                margin: 5px;
                background: #3db5da;
                border-radius: 3px;
            }
            .container{
                display: flex;
                /* display: inline-flex; */
                height: 300px;
            }

Flex Direction Property

ကျွန်တော်ဟာ display value ကို flex or inline flex ကို အသုံးပြုခဲ့ပြီးဖြစ်ပါတယ်။ အခုကျွန်တော်တို့ဟာ flex container ကိုအသုံးပြုပြီးတော့ flex boxes ရဲ့ advantage တွေဖြစ်တဲ့ powerful direction order, alignment and sizing properties တွေကို အသုံးပြုသွားမှာဖြစ်ပါတယ်။ အချို့ flex box properties တွေဟာ flex container ကိုပဲ apply လုပ်ပေးတာဖြစ်ပါတယ်။ သူတို့ဟာ ကျွန်တော်တို့ရဲ့ layout ကိုတော့ apply လုပ်မှာမဟုတ်ပါဘူး။ flex direction property ကိုအသုံးပြုတဲ့ အခါမှာ values တွေဟာ row, column, row-reverse, column-reverse values တွေကို အသုံးပြုနိုင်ပါတယ်။ row နဲ့ column values ရဲ့ layout ကို effect ကတော့ ရှင်းပါတယ်။ row ဆိုရင် ကျွန်တော်တို့ရဲ့ container ထဲမှာရှိတဲ့ items တွေဟာ row အတိုင်း left to right ပေါ်မှာဖြစ်ပါတယ်။ column value ဟာဆိုလည်း items တွေကို top to bottom ပေါ်မှာဖြစ်ပါတယ်။ row-reverse နဲ့ column-reverse ဟာဆိုရင်တော့ row-reverse ဟာဆိုရင် row ရဲ့ ပြောင်းပြန် right to left ဖြစ်ပြီးတော့ column-reverse ဟာဆိုရင် bottom to top ပေါ်မှာဖြစ်ပါတယ်။

Code in CSS

         
            .container{
                display: flex;
                /* display: inline-flex; */
                direction: row;
                
            }

Flex Wrap Property

Flex container ရဲ့ default အရ flex items တွေဟာ single line မှာ နေရာယူမှာဖြစ်ပါတယ်။ အဲ့ဒါကို flex line လို့ခေါ်ပါတယ်။ flex container ဟာဆိုရင် items တွေအားလုံးကို flex line တစ်ခုထဲမှာဘဲ fit ဖြစ်အောင်လုပ်ထားတာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ browser ကို resize လုပ်လိုက်တဲ့အခါမှာ container ဟာ narrow ဖြစ်လာတာကို တွေ့ရပြီး flex items တွေဟာ flex container ရဲ့ boundaries ကို overflow ဖြစ်သွားမှာဖြစ်ပါတယ်။ flex container ကို multiple flex lines နဲ့ items တွေကို break လုပ်ချင်တဲ့အခါမှာ flex wrap property ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ flex wrap property နဲ့ flex container ကို single line or multi-line layout မှာ control လုပ်နိုင်ပါတယ်။ flex wrap property မှာဆိုရင် အသုံးပြုရတဲ့ value ကတော့ wrap ပဲဖြစ်ပါတယ်။ အဲ့ value ကို အသုံးပြုပြီးတော့ flex container ကို browser ကို resize လုပ်တဲ့အခါမှာလည်း အဆင်ပြေအောင် control လုပ်နိုင်ပါတယ်။

Code in CSS

         
            .container{
                display: flex;
                /* display: inline-flex; */
                flex-direction: column;
                flex-wrap: wrap;
                height: 250px;
                
            }

Justity Content Property

ကျွနတော်တို့ဟာ floats or inline block display ကို layout အတွက်အသုံးပြုတဲ့အခါမှာ elements တွေဟာ space ရဲ့ အပြင်ဘက် next line မှာ break ဖြစ်သွားကြပါတယ်။ Flex box ဟာ flexible ဖြစ်ပါတယ်။ သူဟာ available space ကို distribute လုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ elements တွေရဲ့ display value ကို flex or inline flex နဲ့ set လုပ်နိုင်ပါတယ်။ flex items ရဲ့ direction ကိုပဲ set လုပ်နိုင်ပါတာမဟုတ်ပါဘူး flex container ရဲ့ ability တွေဖြစ်တဲ့ flex items ရဲ့ width and height တို့ကိုလည်း change လုပ်နိုင်ပါတယ်။ အဲ့ဒါတွေကို လုပ်ဆာင်ဖို့ရန်အတွက် justified content property ကိုအသုံးပြုနိုင်ပါတယ်။ သူဟာ flex items တွေရဲ့ padding and margins တွေကို accounted လုပ်ပြီးတဲ့အခါမှာ available space ကို distribute လုပ်ပေးပါတယ်။ justified content property ကို flex containers တစ်ခုထဲမှာဘဲအသုံးပြုနိုင်တာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ justify content property မှာ value ကို flex-start လို့ပေးလိုက်တဲ့အခါမှာ ကျွန်တော်တို့ရဲ့ flex items တွေဟာဆိုရင် flex line ရဲ့ start မှာနေရာယူမှာဖြစ်ပါတယ်။ flex-start လိုပဲ flex-end ကိုအသုံးပြုတဲ့အခါမှာ flex line ရဲ့ end မှာ items တွေဟာ နေရာယူမှာဖြစ်ပါတယ်။ နောက်ထပ် justify-content value ကို center ပေးလိုက်တဲ့အခါမှာ ကျွန်တော်တို့ရဲ့ items တွေဟာ container ရဲ့ center မှာနေရာယူမှာဖြစ်ပါတယ်။ center value ကိုအသုံးပြုတဲ့အခါမှာ first item ရဲ့ start edge နဲ့ last item ရဲ့ end edge တွေရဲ့ empty space ဟာဆိုရင် equal ဖြစ်နေမှာဖြစ်ပါတယ်။ နောက်ထပ် အသုံးများတဲ့ value တွေကတော့ space-between နဲ့ space-around ပဲဖြစ်ပါတယ်။ သူတို့ဟာဆိုရင် line တစ်ခုထဲမှာဘဲ available space ကို distribute လုပ်ပေးတာပဲဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် browser ဟာဆိုရင် တစ်ကယ်လိုအပ်တဲ့ space ကို auto တွက်ထုပ်ပေးမှာဖြစ်ပါတယ်။ space-between value ကို အသုံးးပြုလိုက်တဲ့အခါမှာ flex items တွေကြားထဲမှာရှိတဲ့ space ဟာဆိုရင် equal ဖြစ်နေမှာဖြစ်ပါတယ်။ တစ်ကယ်လို့ ကျွန်တော်တို့ဟာ flex items တစ်ခုချင်းစီရဲ့ spacing ကို equal ဖြစ်ချင်တယ်ဆိုရင် space-around value ကိုအသုံးပြုနိုင်ပါတယ်။ ကျန်တော်တို့ဟာ container ရဲ့ background or border ကို set လုပ်ချင်တဲ့အခါမှာ space-around value ဟာဆိုရင် useful ဖြစ်ပါတယ်။ margin ဟာဆိုရင် flex item ပေါ်မှာ significant effect တစ်ခုရှိပါတယ်။ first item ကို margin ကို auto ပေးလိုက်တဲ့အခါမှာ first item နဲ့ remaining items ကြားမှာ extra space ကို distribute လုပ်ပေးမှာဖြစ်ပါတယ်။

Flex Grow Property

ကျွန်တော်တို့ဟာ flex items တွေရဲ့ container ထဲမှာရှိတဲ့ available space ကို flex grow property ကိုအသုံးပြုပြီးတော့ adjust လုပ်နိုင်ပါတယ်။ first, second or ဘယ် flex item ရဲ့ container ထဲမှာ more space ကို ဒီ property ကို အသုံးပြုပြီးတော့ take up လုပ်နိုင်ပါတယ်။ အဲ့မှာဆိုရင် flex grow ရဲ့ default value ဟာဆိုရင် zero ကနေစတာပဲဖြစ်ပါတယ်။ adjust လုပ်ချင်တဲ့အခါမှာ 1,2,3 ဆိုပြီးတော့ numbers တွေကိုအသုံးပြုနိုင်ပါတယ်။ flex grow property ဟာ flex container ဟာ narrow ဖြစ်လာတဲ့အချိန်မှာ flex items တွေရဲ့ width ကို auto adjust လုပ်ပေးသွားမှာဖြစ်ပါတယ်။

Flex Basic Property

Flex basic property ကိုအသုံးပြုတာဟာ ကျွန်တော်တို့ရဲ့ flex items တွေရဲ့ size ပေါ်မှာ ပိုပြီးတော့ control လုပ်နိုင်ပါတယ်။ flex basic property ဟာဆိုရင် display လုပ်တဲ့နေရာမှာ flex grow property နဲ့ equal ဖြစ်ပါတယ်။ flex basic property ကိုသုံးပြုတဲ့အခါမှာ width and height properties ကို same values နဲ့ accept လုပ်နိုင်ပါတယ်။ flex basic property ကို 200 pixel set လုပ်လိုက်တဲ့အခါမှာ flex items တွေအားလုံးရဲ့ size ဟာ 200 pixel မှာပဲ equal ဖြစ်နေမှာဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် flex items တွေမှာဆိုရင် flexible width တွေရှိကြပါတယ်။ ကျွန်တော်တို့ဟာ flex-grow and flex-basic values တွေကို flex ဆိုတဲ့ shorthand property ကိုအသုံးပြုပြီးတော့ ရေးနိုင်ပါတယ်။ flex-shrink property ဟာဆိုရင် flex-grow နဲ့ opposite ဖြစ်ပါတယ်။

                    flex-wrap: wrap;

                    flex-grow: 2;
