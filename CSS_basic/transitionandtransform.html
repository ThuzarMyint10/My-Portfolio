Introductin to Transition and Transform

အခုလေ့လာမှာကတော့ CSS Transition and Transforms အကြောင်းပဲဖြစ်ပါတယ်။ သူတို့ကို developers တွေဟာ simple animations and websites or apps တွေရဲ့ enhance the interactive တွေကို create လုပ်တဲ့နေရာမှာအသုံးပြုပါတယ်။ ကျွန်တော်တို့ဟာ webpages တွေကို browser ပေါ်မှာ ဘယ်လို appear ဖြစ်ချင်တယ်ဆိုတာကို control လုပ်နိုင်ပါတယ်။ ဥပမာ button ရဲ့ color လေးတွေ hover လုပ်လိုက်တဲ့ အချန်မှာ change သွားတာတွေ layout ရဲ့ color တွေchange သွားတာတွေ content တွေကို position လုပ်တာတွေကိုလုပ်ဆောင်နိုင်ပါတယ်။ CSS transitions and transforms နဲ့ content တွေကို animations လေးတွေနဲ့ add လုပ်နိုင်ပါတယ်။ CSS transitions features ဟာဆိုရင် value တစ်ခုကနေ တစ်ခုကို second ပိုင်းတွေအတွင်းမှာ CSS properties တစ်ခုကနေ အများကြီးကို change လုပ်နိုင်ပါတယ်။ CSS transforms features ဟာဆိုရင် element တစ်ခုရဲ့ position ကို change လုပ်နိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ transforms functions တွေဖြစ်တဲ့ move, rotate, scale and skew ဆိုတဲ့ functions တွေကို အသုံးပြုပြီးတော့ page ရဲ့ content ကို control လုပ်နိုင်ပါတယ်။ Transitions and transforms ဟာဆိုရင် CSS features တွေနဲ့ အမှန်တစ်ကယ်တော့ related ဖြစ်နေတာမဟုတ်ပါဘူး။ သူတို့ဟာဆိုရင် stand alone တွေပါဖြစ်ကြပါတယ်။ Element တစ်ခုကို transition မပါဘဲနဲ့ transforms လုပ်နိုင်ပါတယ်။ Site မှာ အမှန်တစ်ကယ် သသေပ်တဲ့ animation effects တွေကို transitioning the position, rotation or scale တွေကို page မှာရှိတဲ့ element တိုင်းကို apply လုပ်ပြီးတော့ addလုပ်နိုင်ပါတယ်။ ဥပမာ images and icons တွေနဲ့ size and positions တွေကို animate ဖြစ်လာအောင်လုပ်တာတွေကို လုပ်ဆောင်နိုင်ပါတယ်။

Transition Duration

CSS transitions ဟာဆိုရင် အချိန်တစ်ခုအတွင်းမှာပေးထားတဲ့ value တတစ်ခုကနေ တစ်ခုကို CSS properties တွေ changing လုပ်ခြင်းဖြင့် interface ရဲ့ parts တွေကို animate ဖြစ်အောင်လုပ်တဲ့ အလွယ်အကူဆုံးနည်းလမ်းပဲဖြစ်ပါတယ်။ Normally အရဆိုရင်တော့ CSS property တစ်ခုရဲ့ value ကို change လုပ်လိုက်တဲ့အခါမှာ အဲ့ change လုပ်တာဟာ ရုတ်တရက်ပြောင်းလဲသွားတာပဲဖြစ်ပါတယ်။ ဥပမာ button တစ်ခုပေါ်မှာ hover လုပ်လိုက်တဲ့အချိန်မှာ သူ့ရဲ့ background color ဟာ change သွားတာမျိုးတို့ပဲဖြစ်ပါတယ်။ Value တစ်ခုကနေ တစ်ခုကို transition လုပ်တဲ့ CSS properties မှာဆိုရင် ဘယ် properties ကို transition လုပ်မယ် ပြီးတော့ အဲ့ transition လေးဟာ ဘယ်လောက်အချိန်ကြာယူမယ်ဆိုတာကို browser ကို ပြောပြဖို့လိုပါတယ်။ Transition တစ်ခုကို အချိန်ဘယ်လောက်ကြမယ်ဆိုတာကိုပြောပြဖို့အတွက် transition duration property ကိုအသုံးပြုမှာဖြစ်ပါတယ်။ သူ့မှာဆိုရင် ကျွန်တော်တို့ဟာ value ကို second or milisecond နဲ့ပေးနိုင်ပါတယ်။ Transitions မှာဆိုရင် hover လုပ်တဲ့အချိန်ပြောင်းသွားတာတွေ mouse click လုပ်လိုက်တဲ့အချိန်မှာ ပြောင်းလဲသွားတာတွေ focus လုပ်ထားတာတဲ့အချိန်မှာ ပြောင်းလဲသွားတာတွေကို လုပ်ဆောင်နိုင်ပါတယ်။

                .photo_container .button{
                    display: flex;
                    justify-content: center;

                    flex-wrap: nowrap;

OverLay Property

Animatable properties တွေပဲ CSS နဲ့ transitioned လုပ်တာပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ CSS property တိုင်းကိုတော့ animate လုပ်လို့မရပါဘူး။ CSS transitions အတွက် အသုံးပြုနိုင်တဲ့ animatable properties အားလုံးကို w3.org မှာ လေ့လာနိုင်ပါတယ်။ အဲ့မှာဆိုရင် CSS နဲ့ transitioned လုပ်နိုင်တဲ့ properties တွေကို listed လုပ်ထားတာပဲဖြစ်ပါတယ်။ Image ပေါ်မှာ hover လုပ်လိုက်တဲ့အခါမှာ image ရဲ့ description and title ဟာ image ပေါ်မှာပေါ်လာဖို့ရန်အတွက်overlay property ကို အသုံးပြုပေးရမှာဖြစ်ပါတယ်။ အဲ့လို အသုံးပြုတဲ့အခါမှာ transition-duration ကိုလည်းပေးခဲ့ရမှာဖြစ်ပါတယ်။

Code in CSS

        
                .photo-overlay {
                    opacity: 0;
                    transition-property: opacity;
                    transition-duration: .4s;
                    background: #171616a3;
                }
                
                .photo-overlay:hover {
                    opacity: 1;
                    cursor: pointer;
                }

Transiton Property

CSS transitions ဟာဆိုရင multiple transitions တွေကို defined လုပ်ဖို့ရန်အတွက် ခွင့်ပြုပေးပါတယ်။ ကျွန်တော်တို့ဟာ button ပေါ်ကို hover လုပ်လိုက်တဲ့အခါမှာ border တွေဖြစ်သွားတာတွေ button ရဲ့ color တွေ change သွားတာတွေကိုလည်း လုပ်ဆောင်နိုင်ပါတယ်။ border-radius, background, and color တို့အတွက်ကိုလည်း transition-duration ကိုတစ်ခါထဲပေးနိုင်ပါတယ်။ Transition ရဲ့ multiple properties တွေကို transition-property ထဲမှာ separate လေးတွေခြားပြီးတော့ ပေးနိုင်ပါတယ်။ Transition property list ထဲမှာရှိတဲ့ first property ဟာ ဆိုရင် transition duration property ထဲက first value နဲ့ match ဖြစ်နေမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ specific property တွေဖြစ်တဲ့ transition property and transition duration နှစ်ခုလုံးနဲ့ transition လုပ်နိုင်ပါတယ်။

Transition Dealy Property

ကျွန်တော်တို့ဟာ CSS properties တွေရဲ့ set of values တစ်ခုကနေစ်ခုကို transition လုပ်တာကိုလေ့လာခဲ့ပြီးဖြစ်ပါတယ်။ အခုတစ်ခုတစ်ခါမှာတာ့ transition တစ်ခုကို start လုပ်လိုက်တဲ့အချိန်မှာ transition property ကိုအသုံးပြုပြီးတော့ ဘယ်လို delay လုပ်နိုင်တယ်ဆိုတာကိုလေ့လာသွားမှာဖြစ်ပါတယ်။ Transition delay property ကိုအသုံးပြုပြီးတော့ transition ဟာ ဘယ်အချိန်မှာစမယ်ဆိုတာကို control လုပ်နိုင်ပါတယ်။ အဲ့ဒါကို create လုပ်ဖို့ဆိုရင် steps တွေအများကြီးနဲ့ လုပ်ဆောင်ရပါတယ်။ ဥပမာ နှစ်ခုထက်ပိုတဲ့ transition တွေကို trigger လုပ်နိုင်ပြီးတော့ အဲ့ transition တစ်ခုချင်းစီးမှာ မတူညီတဲ့ delay time တွေကို sets လုပ်နိုင်ပါတယ်။ Transition delay property မှာ values တွေကို set လုပ်တဲ့အခါမှာ seconds or milliseconds နဲ့ set လုပ်နိုင်ပါတယ်။ Transition-property and transition-duration property တွေပဲ delay values တွေကိုလည်း comma တွေနဲ့ separate လုပ်ပေးရမှာဖြစ်ပါတယ်။

Code in CSS

        
            /* Button Transition */

            .button {
                opacity: 0;
                transition-property: opacity,background,box-shadow;
                transition-duration: .5s;
                transition-delay: .2s, .3s;
            }
            
            .button:hover {
                background: rgba(74,137,202,1);
                box-shadow: 0 0 0  3px rgba(255,255,255, .7);
            }
            
            .photo-overlay:hover .button {
                opacity: 1;
            }

Timing Function Property

Transition တစ်ခုရဲ့ speed ဟာဆိုရင် သူ့ရဲ့ duration ပေါ်မှာ မူတည်ပြီးတော့ ပြောင်းလဲနိုင်ပါတယ်။ Transition speed ကို timing function ကို set လုပ်ပြီးတော့ rapidly or slowly လုပ်နိုင်ပါတယ်။ Timing function ကို easing function လို့လဲခေါ်ပါတယ်။ Real life objects တွေဟာဆိုရင် constant speed မှပဲ move လုပ်ကြတာမဟုတ်ပါဘူး။ Timing functions နဲ့ browser ဟာ real life motion ကို အတုလုပ်နိုင်ပါတယ်။ CSS timing function property ကိုအသုံးပြုပြီးတော့ ကျွန်တော်တို့ဟာ easing effects တွေကို time intervals တွေေပေးပြီးတော့လည်း create လုပ်လို့ရသလို predefined timing function keyword values တွေထဲက တစ်ခုကိုပေးပြီးတော့လည်း create လုပ်နိုင်ပါတယ်။ Keyword values တွေကတော့ ease, linear, ease-in, ease-out, and ease-in-out ဆိုပဲဖြစ်ပါတယ်။ သူတို့ဟာဆိုရင် transition တစ်ခုရဲ့ အစနဲ့ အဆုံးကို ဘယ်လိုဖြစ်စေချင်တယ်ဆိုတာကို ဆုံးဖြတ်ပေးတာပဲဖြစ်ပါတယ်။ Ease ဟာဆိုရင် browser ကနေ apply လုပ်ပေးလိုက်တဲ့ default timing function ပဲဖြစ်ပါတယ်။ Linear timing function မှာဆိုရင်တော့ linear motion ကို maintain လုပ်ပေးပါတယ်။ Ease-in value ဟာ starts မှာ gradually ဖြစ်ပြီးတော့ stops မှာ suddenly ဖြစ်သွားတာဖြစ်ပါတယ်။ Ease-out ဟာဆိုရင် ease-in နဲ့ opposite ဖြစ်တာဖြစ်ပါတယ်။ သူ့မှာဆိုရင် starts ဟာ suddenly ဖြစ်ပြီးတော့ stops မှာ slow down ဖြစ်သွားမှာဖြစ်ပါတယ်။ Ease-in-out ဟာဆိုရင် starts and stops ဟာ gradually ဖြစ်ပါတယ်။ Timing function value တွေဟာဆိုရင် transition duration ကိုတော့ effect ဖြစ်မှာမဟုတ်ပါဘူး။ Timing function value တစ်ုခုချင်းစီဟာဆိုရင် transition property မှာ listed လုပ်ထားတဲ့ corresponding properties တွေကို apply လုပ်ပေးတာပဲဖြစ်ပါတယ်။

Code in CSS

        
            /***************
            TIMING FUNCTION
            ***************/
            .btn-icon{
                opacity: o;
                display: inline-block;
                width: 18px;
                height: 18px;
                background: url(img/thambnalis/download.png);

            }
            .btn-icon {
                transition-property: opacity, left;

Transition Shorthand Property

ကျွန်တော်တို့ဟာ transition properties and values တစ်ခုချင်းစီကို transition property ကိုအသုံးပြုပြီးတော့ ရေးနိုင်ပါတယ်။ ကျွန်တော်တို့ရဲ့ overlay မှာဆိုရင် ရှေ့ဆုံးကတော့ opacity ပဲရေးရမှာဖြစ်ပါတယ်။ နောက်ကတော့ transition duration အတွက် ကိုရေးပေးရမှာဖြစ်ပါတယ်။ အဲ့ဒါတွေကြားထဲမှာတော့ comma နဲ့ separate လုပ်ပေးရမှာဖြစ်ပါတယ်။ အဲ့လိုရေးတဲ့အခါမှာ time values နှစ်ခုပါပါတယ်။ တစ်ခုကတော့ transition duration, နောက်တစ်ခုကတော့ transition delay အတွက်ပဲဖြစ်ပါတယ်။ အဲ့ time values နှစ်ခုရဲ့ ကြားခဲ့ထဲမှာ transition timing function ကို duration and delay values နှစ်ခုကြားထဲမှာ ထည့်ပေးရမှာဖြစ်ပါတယ်။ transition delay နဲ့ timing function ဟာ optional ဖြစ်ပါတယ်။

Transform Property

CSS transform ဟာဆိုရင် elements ရဲ့ size, position and change တွေကို appearance ဖြစ်ဖို့ရန်အတွက် different ways တွေရှိပါတယ်။ Transform နဲ့ content တွေကို rotating, scaling, skewing, and moving တွေလုပ်ခြင်းဖြင့် manipulate လုပ်နိုင်ပါတယ်။ CSS transformations တွေအားလုံးဟာ transform property လို့ခေါ်တဲ့ property တစ်ခုထဲမှာဘဲ လုပ်ဆောင်နိုင်ပါတယ်။ Page မှာ transform ကိုအသုံးပြုဖို့လိုအပ်တဲ့အခါမှာ transform property တစ်ခုထဲမှာဘဲ transform ရဲ့ one or several functions တွေကို လုပ်ဆောင်နိုင်ပါတယ်။ ကျွန်တော်တို့ဟာ transform property ကိုအသုံးပြုတဲ့အခါမှာ rotation function ကိုအသုံးပြုရမှာဖြစ်ပါတယ်။ Rotation function ထဲမှာ arguments တွေကို မတူညီတဲ့ angle units တွေနဲ့ accepts လုပ်နိုင်ပါတယ်။ ဥပမာ degrees, gradients, radians, and turns တို့ပဲဖြစ်ပါတယ်။ သူတို့ထဲမှာအသုံးများတာကတော့ degrees unit ပဲဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ rotation function ကိုအသုံးပြုပြီးတော့ image တွေကို clockwise or counterclockwise ကို control လုပ်နိုင်ပါတယ်။

Code in CSS

        
            /******************
            TRANSFORM:ROTATION
            *******************/

ကျွန်တော်တို့ဟာ element တစ်ခုကို 3D ကို transform မလုပ်ခင်မှာ 3D perspective ကို page မှာ အရင်ဆုံး establish လုပ်ဖို့လိုပါတယ်။ Perspective ဟာဆိုရင် ကျွန်တော်တို့ရဲ့ transform ကို flat and two dimensional မှာ appear ဖြစ်ဖို့ရန်အတွက် ပဲဖြစ်ပါတယ်။ 3D ကို define လုပ်ဖို့ရန်အတွက် perspective property ကိုအသုံးပြုနိုင်ပါတယ်။ Perspective property ကို page ရဲ့ body element, div element, or any other block-level element တွေမှာလည်း apply လုပ်နိုင်ပါတယ်။ Perceptive property ဟာ illusion of depth ကို create လုပ်ပေးတာပဲဖြစ်ပါတယ်။ သူဟာဆိုရင် viewer နဲ့ objects ကြားထဲမှာရှိတဲ့ transforming လုပ်တဲ့ distance ကို set လုပ်ပေးတာပဲဖြစ်ပါတယ်။ Distance ကို ကျွန်တော်တို့ အသုံးပြုလိုက်တဲ့ perspective value နဲ့ determined လုပ်ထားတာပဲဖြစ်ပါတယ်။ Perspective ဟာဆိုရင် value တစ်ခုကိုပဲ accept လုပ်တာပဲဖြစ်ပါတယ်။ အဲ့ value ကို pixels, ems or rems တွေနဲ့ပေးနိုင်ပါတယ်။ Perspective ကို define လုပ်လိုက်တာနဲ့ ကျွန်တော်တို့ဟာ page မှာ ဘယ်လို changes တွေကို တွေ့ရမှာမဟုတ်ပါဘူး။ ဘာကြောင့်လဲဆိုတော့ perspective property ဟာဆိုရင် 3D objects ကို သူတစ်ဦးထဲ create မလုပ်နိုင်တဲ့အတွက်ကြောင့်ပဲဖြစ်ပါတယ်။ အဲ့မှာဆိုရင် transform property ကို add လုပ်ပေးဖို့လိုပါတယ်။

Translate3D Shorthand Property

ကျွန်တော်တို့ဟာ element ကို rotate လုပ်တဲ့အခါမှာ rotate 3D function ကိုအသုံးပြုပြီးတော့ multiple directions မှာ rotate လုပ်နိုင်ပါတယ်။ Translate 3D နဲ့ ကျွန်တော်တို့ဟာ X, Y, and Z axis တွေမှာ translation ကို function တစ်ခုထဲမှာ define လုပ်နိုင်ပါတယ်။ Translate 3D ဟာဆိုရင် translate X, translate Y and translate Z တို့ရဲ့ shorthand function ပဲဖြစ်ပါတယ်။ Values တွေကိုတော့ comma နဲ့ separate လုပ်ပေးရမှာဖြစ်ပါတယ်။ ကျွန်တော်တို့ဟာ project ထဲမှာ CSS ရဲ့ transform function or property ကိုအသုံးပြုမယ်ဆိုရင် CanIUse.com မှာလေ့လာနိုင်ပါတယ်။
